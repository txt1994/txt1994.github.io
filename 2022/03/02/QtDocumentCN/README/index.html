QtDocumentCN

本项目为翻译 Qt 官方文档，内容基于 Qt 最新发布版本 5.15，采用 CC BY-NC-SA 4.0 协议发布。

参与编写#

关于本项目的目录结构、文件管理、格式规范等，详见贡献指南。

完成度追踪表，用于追踪待完成或有待进一步维护的文档状态。

辅助中文文档编写的脚本和工具，详见辅助工具。

技术词汇的翻译可参考 词汇对照表。

关于如何将英文文档信达雅地翻译为中文技术文档，可参考 语法小贴士。

参与者公约#

参与相关讨论时，请遵守参与者公约。

加入社区#

可以通过发起issue的方式建立讨论。

此外，我们组建了 QtDocumentCN-社区组 微信群，可以添加@CryFeiFei 的微信Perfect_Cheung，备注QtDoc，待邀请入群后在群里详细聊。

项目初期有可能有很多不完善的地方，希望各位开发提出不合理的地方。
