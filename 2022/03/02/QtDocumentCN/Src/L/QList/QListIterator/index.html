详细描述#

QList同时具有Java-style iterators和STL-style iterators。STL风格的迭代器更有效，应该是首选。

使用迭代器的一个替代方法是使用索引位置。大多数QList的成员函数将索引作为它们的第一个参数，这使得访问、修改和删除项目都可以不使用迭代器。

QListIterator允许你在QList、QQueue或QStack上进行迭代。如果你想在遍历列表时修改它，请使用QMutableListIterator来代替。

QListIterator的构造函数需要一个QList作为参数。在构造之后，迭代器位于列表的最开始（第一个项目之前）。下面是如何依次迭代所有元素的方法:

QList<float> list;
...
    QListIterator<float> i(list);
while (i.hasNext())
    qDebug() << i.next();

next()函数返回列表中的下一个项目，并使迭代器前进。与STL风格的迭代器不同，Java风格的迭代器指向中间的项目，而不是直接指向项目。第一次调用next()将迭代器推进到第一个和第二个项目之间的位置，并返回第一个项目；第二次调用next()将迭代器推进到第二个和第三个项目之间的位置，并返回第二个项目；等等。

下面是如何以相反的顺序迭代这些元素的方法:

QListIterator<float> i(list);
i.toBack();
while (i.hasPrevious())
    qDebug() << i.previous();

如果你想找到一个特定值的所有出现，在一个循环中使用 findNext() 或 findPrevious() 。

多个迭代器可以在同一个列表上使用。如果列表在QListIterator活动时被修改，QListIterator将继续在原始列表上迭代，忽略被修改的副本。

See also QMutableListIterator and QList::const_iterator.

成员函数文档#

bool QListIterator::findPrevious(const T &value)#

从当前迭代器位置开始向后搜索value。如果找到value，返回true；否则返回false。

调用后，如果找到了value，迭代器被定位在匹配项之前；否则，迭代器被定位在容器的前面。

See also findNext().

bool QListIterator::findNext(const T &value)#

从当前迭代器位置开始向前搜索value。如果找到value，返回 “true”；否则返回 “false”。

调用后，如果找到了value，迭代器被定位在匹配项之后；否则，迭代器被定位在容器的后面。

See also findPrevious().

const T &QListIterator::peekPrevious() const#

在不移动迭代器的情况下返回前一个项目。

在位于容器前端的迭代器上调用这个函数会导致未定义的结果。

See also hasPrevious(), previous(), and peekNext().

const T &QListIterator::previous()#

返回上一个项目并将迭代器向后移动一个位置。

在位于容器前端的迭代器上调用这个函数会导致未定义的结果。

See also hasPrevious(), peekPrevious(), and next().

bool QListIterator::hasPrevious() const#

如果迭代器后面至少有一个项目，即迭代器不在**容器的前面，则返回 “true”；否则返回 “false”。

See also hasNext() and previous().

const T &QListIterator::peekNext() const#

在不移动迭代器的情况下返回下一个项目。

在位于容器后面的迭代器上调用这个函数会导致未定义的结果。

See also hasNext(), next(), and peekPrevious().

const T &QListIterator::next()#

返回下一个项目并使迭代器前进一个位置。

在位于容器后面的迭代器上调用这个函数会导致未定义的结果。

See also hasNext(), peekNext(), and previous().

bool QListIterator::hasNext() const#

如果在迭代器前面至少有一个项目，即迭代器没有在容器的后面，则返回 “true”；否则返回 “false”。

See also hasPrevious() and next().

void QListIterator::toBack()#

将迭代器移到容器的后面（最后一个项目之后）。

See also toFront() and previous().

void QListIterator::toFront()#

将迭代器移到容器的前面（在第一个项目之前）。

See also toBack() and next().

&QListIterator::operator=(const QList &container)#

使得迭代器在list上操作。迭代器被设置为在列表的前面（第一个项目之前）。

See also toFront() and toBack().

QListIterator::QListIterator(const &list)#

构建一个用于遍历list的迭代器。迭代器被设置为在列表的前面（第一个项目之前）。

See also operator=().
