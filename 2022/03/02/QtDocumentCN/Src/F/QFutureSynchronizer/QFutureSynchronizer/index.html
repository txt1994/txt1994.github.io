详细描述#

QFutureSynchronizer是一个模板类，它简化了一个或多个QFuture 对象的同步化。使用addFuture()或setFuture()函数添加期货。futures()函数返回一个期货的列表。使用clearFutures()来从QFutureSynchronizer中删除所有的期货。

waitForFinished()函数会等待所有的期货完成。QFutureSynchronizer的析构器调用waitForFinished()，提供了一种简单的方法来确保在从一个函数返回之前所有的期货已经完成：

void someFunction()
{
    QFutureSynchronizer<void> synchronizer;

    ...

    synchronizer.addFuture(QtConcurrent::run(anotherFunction));
    synchronizer.addFuture(QtConcurrent::map(list, mapFunction));

    return; // QFutureSynchronizer waits for all futures to finish
}

可以使用setCancelOnWait()函数来改变waitForFinished()的行为。调用setCancelOnWait(true)将使waitForFinished()在等待期货完成之前取消所有期货。你可以使用cancelOnWait()函数查询等待时取消功能的状态。

See also QFuture, QFutureWatcher, and Qt Concurrent.

成员函数文档#

QFutureSynchronizer::QFutureSynchronizer(const &future)#

构建一个QFutureSynchronizer，并通过调用addFuture()开始观察future。

See also addFuture().

QFutureSynchronizer::QFutureSynchronizer()#

构建一个QFutureSynchronizer。

QFutureSynchronizer::~QFutureSynchronizer()#

调用waitForFinished()函数，以确保在销毁这个QFutureSynchronizer之前，所有期货都已经完成。

See also waitForFinished().

void QFutureSynchronizer::addFuture(const &future)#

将future添加到管理fulture的列表中。

See also futures().

bool QFutureSynchronizer::cancelOnWait() const#

如果启用了cancel-on-wait功能，返回 “true”；否则返回 “false”。如果启用了cancel-on-wait功能，waitForFinished()函数将在等待fulture完成之前取消所有fulture。

See also setCancelOnWait() and waitForFinished().

void QFutureSynchronizer::clearFutures()#

从这个QFutureSynchronizer中删除所有管理的fulture。

See also addFuture() and setFuture().

< > QFutureSynchronizer::futures() const#

返回所有管理fulture的列表。

See also addFuture() and setFuture().

void QFutureSynchronizer::setCancelOnWait(bool enabled)#

根据enabled参数，启用或禁用等待时取消的功能。如果enabled为真，waitForFinished()函数将在等待期货完成之前取消所有期货。

See also cancelOnWait() and waitForFinished().

void QFutureSynchronizer::setFuture(const &future)#

设置future为这个QFutureSynchronizer所管理的唯一的未来。这是一个方便的函数，它调用waitForFinished()，然后调用clearFutures()，最后将future传给addFuture()。

See also addFuture(), waitForFinished(), and clearFutures().

void QFutureSynchronizer::waitForFinished()#

等待所有fulture完成。如果cancelOnWait()返回 “true”，在等待它们完成之前，每个fulture都被取消。

See also cancelOnWait() and setCancelOnWait().
