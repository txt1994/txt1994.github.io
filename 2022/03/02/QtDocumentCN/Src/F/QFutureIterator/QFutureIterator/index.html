详细描述#

QFuture同时拥有Java风格的迭代器和STL风格的迭代器。Java风格的迭代器比STL风格的迭代器更高级，更容易使用；另一方面，它们的效率略低。

使用迭代器的一个替代方法是使用索引位置。一些QFuture成员函数将索引作为它们的第一个参数，这使得不使用迭代器而访问结果成为可能。

QFutureIterator允许你对QFuture进行迭代。注意，QFuture没有可变的迭代器（与其他Java风格的迭代器不同）。

QFutureIterator构造器接受一个QFuture作为其参数。在构造之后，该迭代器位于结果列表的最开始（即在第一个结果之前）。下面是如何依次迭代所有结果的方法：

QFuture<QString> future;
...
QFutureIterator<QString> i(future);
while (i.hasNext())
    qDebug() << i.next();

next()函数从未来返回下一个结果（如有必要，等待它变得可用），并推进迭代器。与STL风格的迭代器不同，Java风格的迭代器指向之间的结果，而不是直接指向的结果。第一次调用next()将迭代器推进到第一个和第二个结果之间的位置，并返回第一个结果；第二次调用next()将迭代器推进到第二个和第三个结果之间的位置，并返回第二个结果；以此类推。

下面是如何以相反的顺序迭代这些元素的方法:

QFutureIterator<QString> i(future);
i.toBack();
while (i.hasPrevious())
    qDebug() << i.previous();

如果你想找到一个特定值的所有出现，在一个循环中使用findNext()或findPrevious()。

多个迭代器可以在同一个future上使用。如果在一个QFutureIterator活动时，未来被修改，QFutureIterator将继续在原始未来上迭代，而忽略被修改的副本。

See also QFuture::const_iterator and QFuture.

成员函数文档#

QFutureIterator::QFutureIterator(const &future)#

构建一个用于遍历future的迭代器。迭代器被设置为在结果列表的前面（在第一个结果之前）。

See also operator=().

&QFutureIterator::operator=(const QFuture &future)#

使得迭代器对future进行操作。迭代器被设置为在结果列表的前面（在第一个结果之前）。

See also toFront() and toBack().

bool QFutureIterator::findNext(const T &value)#

从当前迭代器位置开始向前搜索value。如果找到value，返回 “true”；否则返回 “false”。

调用后，如果找到了value，迭代器的位置就在匹配结果之后；否则，迭代器的位置就在结果列表的后面。

See also findPrevious().

bool QFutureIterator::findPrevious(const T &value)#

从当前迭代器位置开始向后搜索value。如果找到value，返回 “true”；否则返回 “false”。

调用后，如果找到value，迭代器的位置就在匹配结果之前；否则，迭代器的位置就在结果列表的前面。

See also findNext().

bool QFutureIterator::hasNext() const#

如果在迭代器前面至少有一个结果，例如，迭代器not在结果列表的后面，则返回true；否则返回false。

See also hasPrevious() and next().

bool QFutureIterator::hasPrevious() const#

Returns true if there is at least one result ahead of the iterator, e.g., the iterator is not at the front of the result list; otherwise returns false.

See also hasNext() and previous().

const T &QFutureIterator::next()#

返回下一个结果并使迭代器前进一个位置。

对位于结果列表后面的迭代器调用此函数会导致未定义的结果。

See also hasNext(), peekNext(), and previous().

const T &QFutureIterator::peekNext() const#

在不移动迭代器的情况下返回下一个结果。

对位于结果列表后面的迭代器调用此函数会导致未定义的结果。

See also hasNext(), next(), and peekPrevious().

const T &QFutureIterator::peekPrevious() const#

在不移动迭代器的情况下返回前一个结果。

在位于结果列表前面的迭代器上调用此函数会导致未定义的结果。

See also hasPrevious(), previous(), and peekNext().

const T &QFutureIterator::previous()#

返回上一个结果并将迭代器向后移动一个位置。

在位于结果列表前面的迭代器上调用此函数会导致未定义的结果。

See also hasPrevious(), peekPrevious(), and next().

void QFutureIterator::toBack()#

将迭代器移到结果列表的后面（最后一个结果之后）。

See also toFront() and previous().

void QFutureIterator::toFront()#

将迭代器移到结果列表的前面（第一个结果之前）。

See also toBack() and next().
