详细描述#

图表视图不需要QGraphicsScene对象来工作。要在现有的QGraphicsScene中显示图表，应该使用QChart或QPolarChart类代替。

另请参阅 QChart and QPolarChart .

成员类型文档编制#

[since 6.2] enum QChartView:: RubberBand flags QChartView:: RubberBands#

这个枚举描述了可以应用于矩形缩放区域的不同类型的橡皮筋效果。

常量 值 描述
QChartView::NoRubberBand 0x0 没有指定缩放区域，因此没有启用缩放功能。
QChartView::VerticalRubberBand 0x1 橡皮筋在水平方向上被锁定在图表的尺寸上，可以垂直拉动以指定缩放区域。
QChartView::HorizontalRubberBand 0x2 橡皮筋在垂直方向上被锁定在图表的尺寸上，可以水平拉动以指定缩放区域。
QChartView::RectangleRubberBand 0x3 橡皮筋被固定在被点击的点上，可以垂直和水平地拉动。
QChartView::ClickThroughRubberBand 0x80 上述橡皮筋选择中的一个选项，如果图表项目接受点击，则允许将左键点击传递给图表项目。要选择这个，OR它与橡皮筋选择模式之一。

这个枚举是在Qt 6.2中引入或修改的。

RubberBands类型是QFlags的一个类型定义。它存储了RubberBand值的OR组合。

[override virtual protected] void QChartView:: mouseMoveEvent ( * event )#

重实现： QGraphicsView::mouseMoveEvent (QMouseEvent *event).

如果橡皮筋矩形在event指定的按压事件中被显示出来，事件数据被用来更新橡皮筋的几何形状。否则，默认的QGraphicsView::mouseMoveEvent()实现被调用。

[override virtual protected] void QChartView:: mousePressEvent ( * event )#

重实现： QGraphicsView::mousePressEvent (QMouseEvent *event).

如果按下鼠标左键并启用橡皮筋，则接受事件event并在屏幕上显示橡皮筋。这使用户能够选择缩放区域。

如果其他的鼠标按钮被按下，或者橡皮筋被禁用，事件会被传递给QGraphicsView::mousePressEvent()。

[override virtual protected] void QChartView:: mouseReleaseEvent ( * event )#

重实现： QGraphicsView::mouseReleaseEvent (QMouseEvent *event).

如果释放鼠标左键并启用橡皮筋，则接受事件event，视图被放大到橡皮筋指定的矩形中。如果释放鼠标右键触发了该事件，视图会被放大。

[override virtual protected] void QChartView:: resizeEvent ( * event )#

重实现： QGraphicsView::resizeEvent (QResizeEvent *event).

使用event指定的数据调整和更新图表区域的大小。

QChartView:: rubberBand () const#

返回当前被图表视图使用的橡皮筋标志。

另请参阅 setRubberBand ().

void QChartView:: setChart ( * chart )#

另请参阅 chart ().将当前的图表设置为chart。新图表的所有权被传递给图表视图，之前图表的所有权被释放。

为了避免内存泄漏，必须删除之前的图表。

void QChartView:: setRubberBand (const & rubberBand )#

设置橡皮筋标志为rubberBand。选择的标志决定了缩放的方式。

注意：橡胶带缩放不支持极地图。

另请参阅 rubberBand ().
