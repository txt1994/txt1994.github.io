详细描述#

一个色轴可以被设置为基于传递的梯度显示一个色标。该刻度有刻度线，其标签是基于QXYSeries::colorby方法中传递的数据。

特性文档编制#

autoRange : bool#

该属性持有的属性表明，是否应该从QXYSeries::colorBy方法中传递的值列表中设置范围，或者说从轴本身中获取。

默认值为 true .

访问函数：

bool autoRange () const
void setAutoRange (bool autoRange )

通知程序信号：

void autoRangeChanged (bool autoRange )

max : #

此特性保持轴的最大值。

设置此属性时，必要时对最小值进行调整，以确保范围保持有效。

访问函数：

qreal max () const
void setMax (qreal max )

通知程序信号：

void maxChanged (qreal max )

min : #

此特性保持轴的最小值。

设置此属性时，如有必要，会对最大值进行调整，以确保范围保持有效。

访问函数：

qreal min () const
void setMin (qreal min )

通知程序信号：

void minChanged (qreal min )

size : #

此属性持有色标的大小。

根据对齐方式的不同，该值表示宽度或高度。

访问函数：

qreal size () const
void setSize (const qreal size )

通知程序信号：

void sizeChanged (const qreal size )

tickCount : int#

这个属性持有轴上的刻度线的数量。如果QColorAxis::gridVisible等于true，这表示在图表上绘制多少条网格线。默认值是5，并且这个数字不能小于2。

注意：网格线在QColorAxis中默认是故意不可见的，因为这种类型的轴并不代表几何值。

访问函数：

int tickCount () const
void setTickCount (int count )

通知程序信号：

void tickCountChanged (int tickCount )

[signal] void QColorAxis:: rangeChanged ( min , max )#

当轴上由tickCount指定的刻度线的数量，改变时，发出这个信号。

注意： 通知程序信号对于特性 tickCount .

[virtual] QColorAxis:: ~QColorAxis ()#

销毁对象。

QColorAxis:: gradient () const#

返回当前在色标上使用的梯度。

注意：如果该轴连接到一个系列，该梯度也被[QXYSeries::colorBy]（qxyseries.html#colorBy）方法所使用。

另请参阅 setGradient .

将色标上的梯度设置为gradient。

注意：如果该轴连接到一个系列，该梯度也被[QXYSeries::colorBy]（qxyseries.html#colorBy）方法所使用。

另请参阅 gradient .
