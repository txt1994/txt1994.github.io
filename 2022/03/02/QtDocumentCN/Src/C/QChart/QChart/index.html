详细描述#

QChart 是 QGraphicsWidget可以展示在 QGraphicsScene。它管理着不同类型的系列和其他与图表相关的对象（如图例和坐标轴）的图形表示。为了简单地在布局中显示图表，可以使用方便的类QChartView来代替QChart。此外，直线、样条、面积和散点系列可以通过使用QPolarChart类呈现为极坐标图。

另请参阅 QChartView and QPolarChart .

成员类型文档编制#

enum QChart:: AnimationOption flags QChart:: AnimationOptions#

该枚举描述了图表中启用的动画。

常量 值 描述
QChart::NoAnimation 0x0 图表中的动画被禁用。这是默认值。
QChart::GridAxisAnimations 0x1 在图表中启用了网格轴动画。
QChart::SeriesAnimations 0x2 在图表中启用了系列动画。
QChart::AllAnimations 0x3 所有的动画类型在图表中都被启用。

AnimationOptions类型是QFlags的一个类型化定义。它存储了AnimationOption值的一个OR组合。

enum QChart:: ChartTheme#

这个枚举描述了图表所使用的主题。

主题是一个内置的UI风格相关的设置集合，应用于图表的所有视觉元素，如颜色、笔、刷子和系列的字体，以及轴、标题和图例。图表主题实例](qtcharts-chartthemes-example.html)说明了如何使用主题。

注意：改变主题将覆盖所有先前应用于系列的定制。

常量 值 描述
QChart::ChartThemeLight 0 浅色主题，这是默认主题。
QChart::ChartThemeBlueCerulean 1 青色的蓝色主题。
QChart::ChartThemeDark 2 黑暗的主题。
QChart::ChartThemeBrownSand 3 沙棕色的主题。
QChart::ChartThemeBlueNcs 4 自然色彩系统（NCS）的蓝色主题。
QChart::ChartThemeHighContrast 5 高对比度的主题。
QChart::ChartThemeBlueIcy 6 冰蓝色的主题。
QChart::ChartThemeQt 7 Qt主题。

enum QChart:: ChartType#

这个枚举描述了图表的类型。

常量 值 描述
QChart::ChartTypeUndefined 0 没有定义图表类型。
QChart::ChartTypeCartesian 1 一个笛卡尔图。
QChart::ChartTypePolar 2 一个极地图。

localizeNumbers : bool#

这个属性决定了数字是否被本地化。

当 true 时，所有出现在各种系列和轴标签中的生成的数字将使用QLocale设置的locale属性来进行本地化。当 “false “时，总是使用C语言。默认为 “false”。

注意：这个属性不影响QDateTimeAxis标签，它总是使用QLocale设置的locale属性。

void QChart:: addSeries ( * series )#

将系列series添加到图表中，并取得它的所有权。

注意：新添加的系列默认不附加任何轴，甚至那些在系列被添加到图表之前可能已经使用createDefaultAxes()为图表创建的轴。如果在显示图表之前没有为新添加的系列附加任何轴，那么该系列将被绘制为具有完全适合该系列的轴的范围，以适应图表的绘图区域。如果同一图表还显示其他有适当附加轴的系列，这可能会引起混淆，所以一定要确保在添加系列后调用createDefaultAxes()或明确为系列附加轴。

另请参阅 removeSeries (), removeAllSeries (), createDefaultAxes ()，和 QAbstractSeries::attachAxis ().

void QChart:: createDefaultAxes ()#

根据已经添加到图表中的系列，为图表创建坐标轴。之前添加到图表中的任何坐标轴将被删除。

注意：此函数必须在所有系列都被添加到图表中后调用。由该函数创建的坐标轴不会自动附加到在该函数被调用后添加到图表中的任何系列上。一个没有附加轴的系列，默认情况下将利用图表的整个绘图区域进行缩放，如果还有其他具有适当附加轴的系列存在，这可能会引起混淆。

Series type Horizontal axis (X) Vertical axis (Y)
QXYSeries QValueAxis QValueAxis
QBarSeries QBarCategoryAxis QValueAxis
QPieSeries None None

如果有几个QXYSeries衍生的系列被添加到图表中，并且没有添加其他类型的系列，那么就只创建一对轴。如果有几个不同类型的系列被添加到图表中，那么每个系列都有自己的轴对。

通过提供系列作为axes()函数调用的参数，以后可以从图表中获得该系列的特定坐标轴。QPieSeries不创建任何坐标轴。

另请参阅 axes () 和 QAbstractSeries::attachAxis ().

void QChart:: zoom ( factor )#

按自定义系数系数放大视图。

一个超过1.0的系数会放大到视图中，一个在0.0和1.0之间的系数会缩小到视图之外。

void QChart:: zoomIn ()#

将视图放大2倍。

void QChart:: zoomIn (const & rect )#

将视图放大到矩形rect仍然完全可见的最大水平。

注意：应用缩放可能会修改附加轴的属性，例如QAbstractAxis::min和QAbstractAxis::max。

注意：这不支持极地图。

void QChart:: zoomOut ()#

将视图放大2倍。

注意：如果结果会包含一个无效的对数轴范围，这将不会有任何作用。

void QChart:: zoomReset ()#

将系列域重置为调用任何缩放方法之前的状态。

注意：这也将重置在第一次缩放操作和调用此方法之间指定的滚动和显式轴范围设置。如果没有进行缩放操作，这个方法不做任何事情。
