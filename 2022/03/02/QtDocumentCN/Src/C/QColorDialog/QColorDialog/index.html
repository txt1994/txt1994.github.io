详细描述#

颜色对话框的功能是让用户选择颜色。例如，你可以在一个绘图程序中使用它，让用户设置画笔的颜色。

静态函数提供模态颜色对话框。

静态 getColor()函数显示对话框，并允许用户指定颜色。这个函数也可以用来让用户选择具有一定透明度的颜色：将ShowAlphaChannel选项作为附加参数传给用户。

用户可以存储 customCount ()不同的自定义颜色。自定义颜色被所有颜色对话框共享，并在程序执行过程中被记住。使用setCustomColor()来设置自定义颜色，使用customColor()来获取它们。

当按下 “挑选屏幕颜色 “按钮时，光标会变成一个十字形，屏幕上的颜色被扫描出来。用户可以通过点击鼠标或回车键选取一种。按Escape键可以恢复进入此模式前最后选择的颜色。

标准对话框 范例展示如何使用 QColorDialog 及其它内置 Qt 对话框。

另请参阅 QColor , QFileDialog , QFontDialog ，和 标准对话框范例 .

成员类型文档编制#

enum QColorDialog:: ColorDialogOption flags QColorDialog:: ColorDialogOptions#

这个枚举指定了影响颜色对话框的外观和感觉的各种选项。

常量 值 描述
QColorDialog::ShowAlphaChannel 0x00000001 允许用户选择颜色的α成分。
QColorDialog::NoButtons 0x00000002 不显示 OK and Cancel buttons. (Useful for “live dialogs”.)
QColorDialog::DontUseNativeDialog 0x00000004 使用Qt的标准颜色对话框，而不是操作系统的本地颜色对话框。

ColorDialogOptions 类型是 typedef 对于 QFlags . It stores an OR combination of ColorDialogOption values.

另请参阅 options , setOption (), testOption ()，和 windowModality ().

currentColor : #

此属性保持当前在对话框中选择的颜色。

访问函数：

QColor currentColor () const

通知程序信号：

options : #

此属性保存了影响对话框外观和感觉的各种选项。

默认情况下，所有选项是被禁用的。

选项应在显示对话框之前设置。在对话框可见的情况下设置它们，并不能保证对对话框立即产生影响（取决于选项和平台）。

访问函数：

另请参阅 setOption () 和 testOption ().

[signal] void QColorDialog:: currentColorChanged (const & color )#

只要对话框中的当前颜色发生变化，就会发出这个信号。当前的颜色是由color指定的。

注意： 通知程序信号对于特性 currentColor .

另请参阅 color and colorSelected ().

[static] int QColorDialog:: customCount ()#

返回QColorDialog所支持的自定义颜色的数量。所有颜色对话框共享相同的自定义颜色。

[override virtual protected] void QColorDialog:: done ( int result )#

重实现： QDialog::done (int r).

关闭对话框并将其结果代码设为 result . 如果用 exec ()显示这个对话框，done()会使本地事件循环结束，exec ()会返回result。

另请参阅 QDialog::done ().

[static] QColorDialog:: getColor (const & initial = Qt::white, * parent = nullptr, const & title = QString(), options = ColorDialogOptions())#

弹出一个具有给定窗口标题的模态颜色对话框（如果没有指定，则为 “选择颜色”），让用户选择一个颜色，并返回该颜色。该颜色最初被设置为initial。该对话框是parent*的一个子程序。如果用户取消对话，它将返回一个无效的（见QColor::isValid()）颜色。

options 自变量允许您定制对话框。

void QColorDialog:: open ( * receiver , const char * member )#

打开对话框并连接其 colorSelected () 信号到槽，指定通过 receiver and member .

信号将断开槽连接，当对话框被关闭时。

QColorDialog:: selectedColor () const#

返回用户通过点击OK或同等按钮所选择的颜色。

注意：这个颜色并不总是与currentColor属性所持有的颜色相同，因为用户在最终选择要使用的颜色之前可以选择不同的颜色。

[static] void QColorDialog:: setCustomColor ( int index , color )#

设置自定义颜色在 index 到 QColor color 值。

注意：此功能不适用于macOS平台上的Native Color Dialog。如果你仍然需要这个函数，请使用QColorDialog::DontUseNativeDialog选项。

另请参阅 customColor ().

void QColorDialog:: setOption ( option , bool on = true)#

设置给定 option 为被启用若 on 为 true；否则，清零给定 option .

另请参阅 options and testOption ().

[static] void QColorDialog:: setStandardColor ( int index , color )#

设置标准颜色在 index 到 QColor color 值。

注意： 这个函数不适用于macOS平台上的Native Color Dialog。如果你仍然需要这个函数，请使用QColorDialog::DontUseNativeDialog选项。

另请参阅 standardColor ().

[override virtual] void QColorDialog:: setVisible ( bool visible )#

重实现： QDialog::setVisible (bool visible).

更改对话框的可见性。若 visible 为 true，展示对话框；否则，被隐藏。

[static, since 5.0] QColorDialog:: standardColor ( int index )#

返回标准颜色在给定 index 作为 QColor 值。

另请参阅 setStandardColor ().

bool QColorDialog:: testOption ( option ) const#

返回 true 若给定 option 被启用；否则，返回 false。

另请参阅 options and setOption ().
