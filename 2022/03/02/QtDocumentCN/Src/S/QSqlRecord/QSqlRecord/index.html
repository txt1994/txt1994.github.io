详细描述#

QSqlRecord 类封装了数据库记录（通常是数据库中表或视图中的一行）的功能和特征。 QSqlRecord 支持添加和删除字段以及设置和检索字段值。

记录字段的值可以通过名称或位置使用 setValue();如果要将字段设置为空，请使用 setNull()。要按名称查找字段的位置，请使用 indexOf()，要查找特定位置的字段名称，请使用 fieldName()。使用 field() 检索给定字段的 QSqlField 对象。使用 contains() 查看记录是否包含特定字段名称。

当生成要在数据库上执行的查询时，只有那些 isGenerated() 为 true 的字段才会包含在生成的 SQL 中。

一条记录的字段可以用 append() 或 insert() 添加，用 replace() 替换，用 remove( ）。可以使用 clear() 删除所有字段。字段数由 count() 给出；可以使用 clearValues() 将它们的所有值清除（为空）。

See also QSqlField and QSqlQuery::record().

销毁对象并释放所有分配的资源。

void QSqlRecord::append(const &field)#

将字段 field 的副本附加到记录的末尾。

See also insert(), replace(), and remove().

void QSqlRecord::clear()#

删除所有记录的字段。

See also clearValues() and isEmpty().

void QSqlRecord::clearValues()#

清除记录中所有字段的值，并将每个字段设置为空。

See also setValue().

bool QSqlRecord::contains(const &name) const#

如果记录中有一个名为 name 的字段，则返回 true； 否则返回false。

int QSqlRecord::count() const#

返回记录中的字段数。

另请参阅 isEmpty()。

QSqlRecord::field(int index) const#

返回位置 index 处的字段。如果 index 超出范围，则函数返回 default-constructed value。

QSqlRecord::field(const &name) const#

这是一个重载函数。

返回名为 name 的字段。

QSqlRecord::fieldName(int index) const#

返回位置 index 的字段名称。如果该字段不存在，则返回一个空字符串。

另见 indexOf()。

int QSqlRecord::indexOf(const &name) const#

返回记录中名为 name 的字段的位置，如果找不到，则返回 -1。字段名称不区分大小写。如果多个字段匹配，则返回第一个。

另见 fieldName()。

void QSqlRecord::insert(int pos, const &field)#

在记录中的 pos 位置插入字段 field。

另请参阅 append()、replace() 和 remove()。

bool QSqlRecord::isEmpty() const#

如果记录中没有字段，则返回 true；否则返回false。

另请参阅 append()、insert() 和 clear()。

bool QSqlRecord::isGenerated(const &name) const#

如果记录有一个名为 name 的字段并且要生成该字段（默认），则返回 true；否则返回false。

另请参阅 setGenerated()。

bool QSqlRecord::isGenerated(int index) const#

这是一个重载函数。

如果记录在 index 位置有一个字段并且要生成该字段（默认），则返回 true；否则返回false。

另请参阅 setGenerated()。

bool QSqlRecord::isNull(const &name) const#

如果名为 name 的字段为空或没有名为 name 的字段，则返回 true；否则返回false。

另请参阅 setNull()。

bool QSqlRecord::isNull(int index) const#

这是一个重载函数。

如果字段 index 为 null 或位置 index 处没有字段，则返回 true；否则返回false。

[since 5.1] QSqlRecord::keyValues(const &keyFields) const#

返回包含 keyFields 中表示的字段的记录，这些字段设置为按字段名称匹配的值。

该函数在 Qt 5.1 引入。

void QSqlRecord::remove(int pos)#

删除位置 pos 处的字段。如果 pos 超出范围，则不会发生任何事情。

另请参阅 append()、insert() 和 replace()。

void QSqlRecord::replace(int pos, const &field)#

用给定的 field 替换位置 pos 的字段。如果 pos 超出范围，则不会发生任何事情。

另请参阅 append()、insert() 和 remove()。

void QSqlRecord::setGenerated(const &name, bool generated)#

将名为 name 的字段的生成标志设置为 generated。如果该字段不存在，则不会发生任何事情。例如，只有 generated 设置为 true 的字段才会包含在由 QSqlQueryModel 生成的 SQL 中。

另请参阅 isGenerated()。

void QSqlRecord::setGenerated(int index, bool generated)#

这是一个重载函数。

将字段 index 的生成标志设置为 generated。

另请参阅 isGenerated()。

void QSqlRecord::setNull(int index)#

将字段 index 的值设置为 null。如果该字段不存在，则不会发生任何事情。

另请参阅 isNull() 和 setValue()。

void QSqlRecord::setNull(const &name)#

这是一个重载函数。

将名为 name 的字段的值设置为 null。如果该字段不存在，则不会发生任何事情。

void QSqlRecord::setValue(int index, const &val)#

将位置 index 处的字段值设置为 val。如果该字段不存在，则不会发生任何事情。

See also value() and setNull().

void QSqlRecord::setValue(const &name, const &val)#

这是一个重载函数。

将名为 name 的字段的值设置为 val。如果该字段不存在，则不会发生任何事情。

QSqlRecord::value(int index) const#

返回记录中位于 index 位置的字段的值。如果 index 超出范围，则返回无效的 QVariant。

另请参阅 setValue()、fieldName() 和 isNull()。

QSqlRecord::value(const &name) const#

这是一个重载函数。

返回记录中名为 name 的字段的值。如果字段 name 不存在，则返回无效变体。

另见 indexOf()。

bool QSqlRecord::operator!=(const &other) const#

bool QSqlRecord::operator==(const &other) const#

如果此对象与 other 相同（即，具有相同顺序的相同字段），则返回 true；否则返回false。

See also operator!=().
