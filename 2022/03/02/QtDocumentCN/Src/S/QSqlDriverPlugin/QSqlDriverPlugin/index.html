详细描述#

SQL 驱动插件是一个简单的插件接口，可以很容易地创建您自己的可以被 Qt 动态加载的 SQL 驱动插件。

编写 SQL 插件是通过继承这个基类，重新实现纯虚函数 create()，并使用 Q_PLUGIN_METADATA() 宏导出类来实现的。 请参阅 Qt 附带的 SQL 插件示例实现（在源代码分发的 plugins/src/sqldrivers 子目录中）。

包含插件元数据的 json 文件包含指示支持的 sql 驱动程序的键列表

{ "Keys": [ "mysqldriver" ] }

See also How to Create Qt Plugins.

成员函数文档#

QSqlDriverPlugin::QSqlDriverPlugin( *parent = nullptr)#

构造一个 SQL 驱动插件并将父级设置为 parent。 这由导出插件的 moc 生成的代码自动调用。

[virtual]QSqlDriverPlugin::~QSqlDriverPlugin()#

销毁 SQL 驱动程序插件。

你永远不必明确地调用它。 当不再使用插件时，Qt 会自动销毁它。

[pure virtual] *QSqlDriverPlugin::create(const &key)#

为名为 key 的驱动程序创建并返回一个 QSqlDriver 对象。 驱动程序键通常是所需驱动程序的类名。 键是区分大小写的。

See also How to Create Qt Plugins.
