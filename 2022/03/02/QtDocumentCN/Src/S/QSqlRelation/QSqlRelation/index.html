详细描述#

QSqlRelation 是 QSqlRelationalTableModel 的辅助类。 有关详细信息，请参阅 QSqlRelationalTableModel::setRelation() 和 QSqlRelationalTableModel::relation()。

See also QSqlRelationalTableModel, QSqlRelationalDelegate, and Relational Table Model Example.

成员函数文档#

QSqlRelation::QSqlRelation(const &tableName, const &indexColumn, const &displayColumn)#

构造一个 QSqlRelation 对象，其中 tableName 是外键引用的 SQL 表名，indexColumn 是外键，displayColumn 是应该呈现给用户的字段。

See also tableName(), indexColumn(), and displayColumn().

QSqlRelation::QSqlRelation()#

构造一个无效的 QSqlRelation 对象。

对于这样的对象，tableName()、indexColumn() 和 displayColumn() 函数返回一个空字符串。

See also isValid().

QSqlRelation::displayColumn() const#

从表 tableName() 返回应该呈现给用户而不是外键的列。

QSqlRelation::indexColumn() const#

从表 tableName() 返回外键引用的索引列。

bool QSqlRelation::isValid() const#

如果 QSqlRelation 对象有效，则返回 true； 否则返回“假”。

void QSqlRelation::swap( &other)#

将 this 与 other 交换。

QSqlRelation::tableName() const#

返回外键引用的表的名称。
