详细描述#

index 指的是数据库中的单个表或视图。 有关构成索引的字段的信息可用于生成 SQL 语句。

成员函数文档#

QSqlIndex::QSqlIndex(const &other)#

构造 other 的副本。

QSqlIndex::QSqlIndex(const &cursorname = QString(), const &name = QString())#

使用游标名称 cursorname 和索引名称 name 构造一个空索引。

&QSqlIndex::operator=(const &other)#

设置索引等于 other。

QSqlIndex::~QSqlIndex()#

销毁对象并释放所有分配的资源。

void QSqlIndex::append(const &field)#

将字段 field 附加到索引字段列表中。 该字段附加了升序排序顺序。

void QSqlIndex::append(const &field, bool desc)#

这是一个过载功能。

将字段 field 附加到索引字段列表中。 除非 desc 为真，否则该字段会附加升序排序。

QSqlIndex::cursorName() const#

返回与索引关联的游标的名称。

See also setCursorName().

bool QSqlIndex::isDescending(int i) const#

如果索引中的字段 i 按降序排序，则返回 true； 否则返回“假”。

QSqlIndex::name() const#

返回索引的名称。

See also setName().

void QSqlIndex::setCursorName(const &cursorName)#

将与索引关联的游标的名称设置为 cursorName。

See also cursorName().

void QSqlIndex::setDescending(int i, bool desc)#

如果 desc 为真，则字段 i 按降序排序。 否则，字段 i 将按升序排序（默认）。 如果该字段不存在，则不会发生任何事情。

See also isDescending().

void QSqlIndex::setName(const &name)#

将索引的名称设置为 name。

See also name().
