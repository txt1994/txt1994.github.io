详细描述#

它最常用于通过定期调用插槽来为 GUI 控件设置动画。您可以通过将其持续时间（以毫秒为单位）传递给 QTimeLine 的构造函数来构建时间线。时间线的持续时间描述了动画将运行多长时间。然后通过调用setFrameRange ()设置合适的帧范围。最后将frameChanged () 信号连接到您希望设置动画的小部件中的合适插槽（例如，QProgressBar中的setValue () ）。当你继续调用start () 时，QTimeLine 将进入 Running 状态，并开始发射frameChanged() 定期，导致您的小部件的连接属性的值以稳定的速率从下端增长到帧范围的上限。您可以通过调用setUpdateInterval ()来指定更新间隔。完成后，QTimeLine 进入NotRunning状态，并发出finished ()。

例子：

...
progressBar = new QProgressBar(this);
progressBar->setRange(0, 100);

// Construct a 1-second timeline with a frame range of 0 - 100
QTimeLine *timeLine = new QTimeLine(1000, this);
timeLine->setFrameRange(0, 100);
connect(timeLine, &QTimeLine::frameChanged, progressBar, &QProgressBar::setValue);

// Clicking the push button will start the progress bar animation
pushButton = new QPushButton(tr("Start animation"), this);
connect(pushButton, &QPushButton::clicked, timeLine, &QTimeLine::start);
...

默认情况下，时间线从头到尾运行一次，您必须再次调用start () 才能从头重新开始。要使时间线循环，您可以调用setLoopCount ()，传递时间线在完成之前应该运行的次数。也可以通过调用setDirection ()改变方向，使时间线向后运行。您还可以通过调用setPaused () 在时间线运行时暂停和取消暂停时间线。对于交互控制，提供了setCurrentTime ()函数，直接设置时间线的时间位置。虽然在NotRunning状态下最有用（例如，连接到QSlider），这个函数可以随时调用。

框架接口对标准小部件很有用，但 QTimeLine 可用于控制任何类型的动画。QTimeLine 的核心在于valueForTime () 函数，它为给定时间生成一个介于 0 和 1 之间的值。该值通常用于描述动画的步骤，其中 0 是动画的第一步，1 是最后一步。运行时，QTimeLine 通过调用valueForTime () 并发出valueChanged () 来生成 0 和 1 之间的值。默认情况下，valueForTime () 应用插值算法来生成这些值。您可以通过调用setEasingCurve () 从一组预定义的时间线算法中进行选择。

请注意，默认情况下，QTimeLine 使用QEasingCurve::InOutSine，它提供一个缓慢增长的值，然后稳定增长，最后缓慢增长。对于自定义时间线，您可以重新实现valueForTime ()，在这种情况下，QTimeLine 的easingCurve属性将被忽略。

另见QProgressBar和QProgressDialog。

成员类型文档#

enum QTimeLine::Direction#

此枚举描述处于运行状态时时间线的方向。

持续的 价值 描述
QTimeLine::Forward 0 时间线的当前时间随时间增加（即，从 0 移动到结束/持续时间）。
QTimeLine::Backward 1 时间线的当前时间随时间减少（即，从结束/持续时间移动到 0）。

See also setDirection().

enum QTimeLine::State#

这个枚举描述了时间线的状态。

持续的 价值 描述
QTimeLine::NotRunning 0 时间线未运行。这是QTimeLine的初始状态，完成后状态QTimeLine重新进入。当前时间、帧和值保持不变，直到调用setCurrentTime () 或调用start () 启动时间线。
QTimeLine::Paused 1 时间线被暂停（即，暂时暂停）。调用setPaused (false) 将恢复时间线活动。
QTimeLine::Running 2 时间线正在运行。当控制处于事件循环中时，QTimeLine将定期更新其当前时间，并在适当的时候发出valueChanged () 和frameChanged ()。

See also state() and stateChanged().

属性文档#

[bindable]currentTime : int#

此属性保存时间线的当前时间。

当QTimeLine处于运行状态时，该值会根据时间线的持续时间和方向不断更新。否则，它是最后一次调用stop () 时的当前值，或者由 setCurrentTime() 设置的值。

注意：您可以将其他属性绑定到 currentTime，但不建议对其设置绑定。随着动画的进行，currentTime 会自动更新，从而取消其绑定。

默认情况下，此属性包含值 0。

[bindable]direction : #

注意：此属性支持QProperty绑定。

当QTimeLine处于Running状态时，此属性保存时间线的方向。

该方向指示时间是从 0 移动到时间线持续时间，还是从持续时间的值移动到调用start () 后的 0。

任何方向绑定不仅会被 setDirection() 删除，还会被toggleDirection () 删除。

默认情况下，此属性设置为Forward。

[bindable]duration : int#

注意：此属性支持QProperty绑定。

此属性以毫秒为单位保存时间线的总持续时间。

默认情况下，此值为 1000（即 1 秒），但您可以通过将持续时间传递给QTimeLine的构造函数或调用 setDuration() 来更改此值。持续时间必须大于 0。

注意：更改持续时间不会导致当前时间重置为零或新的持续时间。您还需要使用所需的值调用setCurrentTime ()。

[bindable]easingCurve : #

注意：此属性支持QProperty绑定。

指定时间线将使用的缓动曲线。如果重新实现valueForTime ()，则忽略此值。

另请参见valueForTime ()。

[bindable]loopCount : int#

注意：此属性支持QProperty绑定。

此属性保存时间线在完成之前应循环的次数。

循环计数为 0 意味着时间线将永远循环。

默认情况下，此属性包含值 1。

[bindable]updateInterval : int#

注意：此属性支持QProperty绑定。

此属性保存每次QTimeLine更新其当前时间之间的时间（以毫秒为单位）。

更新当前时间时，如果当前值发生变化， QTimeLine将发出valueChanged ()，如果帧发生变化，则发出frameChanged ()。

默认情况下，间隔为 40 毫秒，对应于每秒 25 次更新的速率。

成员函数文档#

QTimeLine::QTimeLine(int duration = 1000, *parent = nullptr)#

构造一个持续duration 为毫秒的时间线。parent被传递给QObject的构造函数。默认持续时间为 1000 毫秒。

[private signal]void QTimeLine::finished()#

此信号在QTimeLine完成时发出（即到达其时间线的末端），并且不循环。

注意：这是一个私人信号。它可以用于信号连接，但不能由用户发出。

[private signal]void QTimeLine::frameChanged(int frame)#

QTimeLine在运行状态时会定期发出此信号，但前提是当前帧发生变化。frame是当前帧号。

注意：这是一个私人信号。它可以用于信号连接，但不能由用户发出。

另见QTimeLine::setFrameRange () 和QTimeLine::updateInterval。

[slot]void QTimeLine::resume()#

从当前时间恢复时间线。QTimeLine会重新进入 Running 状态，一旦进入事件循环，它会定期更新其当前时间、帧和值。

与start () 相比，此函数在恢复之前不会重新启动时间线。

另请参见start ()、updateInterval ()、frameChanged () 和valueChanged ()。

[slot]void QTimeLine::setPaused(bool paused)#

如果paused为真，时间线暂停，导致QTimeLine进入 Paused 状态。在调用start () 或 setPaused(false)之前不会发出任何更新信号。如果paused为 false，时间线将恢复并从其离开的地方继续。

另见state () 和start ()。

[slot]void QTimeLine::start()#

启动时间线。QTimeLine将进入 Running 状态，一旦进入事件循环，它会定期更新其当前时间、帧和值。默认间隔为 40 毫秒（即每秒 25 次）。您可以通过调用setUpdateInterval ()来更改更新间隔。

时间线将从位置 0 开始，如果向后移动，则结束。如果你想恢复一个停止的时间线而不重新启动，你可以调用resume () 代替。

另请参见resume ()、updateInterval ()、frameChanged () 和valueChanged ()。

[private signal]void QTimeLine::stateChanged( newState)#

每当QTimeLine的状态发生变化时，都会发出此信号。新状态是newState。

注意：这是一个私人信号。它可以用于信号连接，但不能由用户发出。

[slot]void QTimeLine::stop()#

停止时间线，导致QTimeLine进入NotRunning状态。

See also start().

[slot]void QTimeLine::toggleDirection()#

切换时间线的方向。如果方向是向前，则变为向后，反之亦然。

现有的方向绑定被删除。

See also setDirection().

[private signal]void QTimeLine::valueChanged( value)#

QTimeLine在运行状态时会定期发出此信号，但前提是当前值发生变化。value是当前值。value是一个介于 0.0 和 1.0 之间的数字

注意：这是一个私人信号。它可以用于信号连接，但不能由用户发出。

另请参阅QTimeLine::setDuration ()、QTimeLine::valueForTime () 和QTimeLine::updateInterval。

破坏时间线。

See also valueForTime() and currentFrame().

int QTimeLine::endFrame() const#

返回结束帧，即时间轴结束对应的帧（即当前值为 1 的帧）。

See also setEndFrame() and setFrameRange().

int QTimeLine::frameForTime(int msec) const#

返回与时间msec对应的帧。该值是根据valueForTime ()返回的值，使用开始帧和结束帧的线性插值计算得出的。

另请参见valueForTime () 和setFrameRange ()。

See also valueForTime() and setFrameRange().

void QTimeLine::setEndFrame(int frame)#

将结束帧，即与时间线结束对应的帧（即当前值为 1 的帧）设置为frame。

See also endFrame(), startFrame(), and setFrameRange().

void QTimeLine::setFrameRange(int startFrame, int endFrame)#

将时间线的帧计数器设置为从startFrame、end 和endFrame 开始。对于每一个时间值，当你调用currentFrame ()或frameForTime ()时， QTimeLine会使用valueForTime ()的返回值，通过插值找到对应的帧。

当处于运行状态时，QTimeLine也会在帧发生变化时发出frameChanged () 信号。

See also startFrame(), endFrame(), start(), and currentFrame().

void QTimeLine::setStartFrame(int frame)#

将起始帧设置为 frame ，即与时间线的开始相对应的帧（即当前值为 0 的帧） 。

See also startFrame(), endFrame(), and setFrameRange().

int QTimeLine::startFrame() const#

返回起始帧，即时间轴开始对应的帧（即当前值为 0 的帧）。

See also setStartFrame() and setFrameRange().

QTimeLine::state() const#

返回时间线的状态。

See also start(), setPaused(), and stop().

[virtual] QTimeLine::valueForTime(int msec) const#

返回时间msec的时间线值。返回值根据曲线形状而变化，始终介于 0 和 1 之间。如果msec为 0，则默认实现始终返回 0。
