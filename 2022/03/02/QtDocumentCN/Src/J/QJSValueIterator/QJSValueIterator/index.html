详细描述#

QJSValueIterator构造器需要一个[QJSValue]（）作为参数。在构造之后，迭代器位于属性序列的最开始。下面是如何在QJSValue的所有属性中进行迭代:

QJSValue object;
...
QJSValueIterator it(object);
while (it.hasNext()) {
    it.next();
    qDebug() << it.name() << ": " << it.value().toString();
}

next()使迭代器前进。name() 和 value() 函数返回被跳过的最后一个项目的名称和值。

请注意，QJSValueIterator只迭代QJSValue自身的属性；也就是说，它不遵循原型链。你可以使用像这样的一个循环来遵循原型链:

QJSValue obj = ...; // the object to iterate over
while (obj.isObject()) {
    QJSValueIterator it(obj);
    while (it.hasNext()) {
        it.next();
        qDebug() << it.name();
    }
    obj = obj.prototype();
}

另请参阅 QJSValue::property ().

QJSValueIterator:: QJSValueIterator (const & object )#

构造迭代器为遍历 object 。迭代器被设置为在属性序列的前面（第一个属性之前）。

&QJSValueIterator:: operator= ( & object )#

使得迭代器对object 进行操作。迭代器被设置为在属性序列的前面（第一个属性之前）。

QJSValueIterator:: ~QJSValueIterator ()#

销毁迭代器。

bool QJSValueIterator:: hasNext () const#

如果在迭代器前面至少有一个项目，则返回true（即迭代器不是在属性序列的后面）；否则返回false。

另请参阅 next ().

QJSValueIterator:: name () const#

返回使用next()跳过的最后一个属性的名称。

另请参阅 value ().

bool QJSValueIterator:: next ()#

使迭代器前进一个位置。如果至少有一个项目在迭代器前面，则返回真（即迭代器不是在属性序列的后面）；否则返回假。

另请参阅 hasNext () 和 name ().

QJSValueIterator:: value () const#

返回使用next()跳过的最后一个属性的值。

另请参阅 name ().
