详细描述#

一个JSON对象是一个键值对的列表，其中键是唯一的字符串，值由QJsonValue表示。

QJsonObject可以被转换成QVariantMap，也可以从QVariantMap中转换。你可以用size()、insert()和remove()条目来查询(key, value)对的数量，并使用标准的C++迭代器模式来迭代其内容。

QJsonObject是一个隐式共享的类，只要它没有被修改，就与它所创建的文档共享数据。

你可以通过QJsonDocument将该对象转换成基于文本的JSON。

另请参阅 在 Qt 中支持 JSON and JSON 保存游戏范例 .

[static, since 5.5] QJsonObject:: fromVariantHash (const QVariantHash & hash )#

转换变体哈希*到QJsonObject。

hash中的键将被用作JSON对象中的键，而QVariant的值将被转换为JSON值。

注意：从QVariant的转换不是完全无损的。请参阅 QJsonValue::fromVariant () 中的文档 了解更多信息。

该函数在 Qt 5.5 引入。

另请参阅 fromVariantMap (), toVariantHash ()，和 QJsonValue::fromVariant ().

QJsonObject:: insert (const & key , const & value )#

插入一个新的项目，其键为key，其值为value。

如果已经有一个键为key的项目，那么这个项目的值将被替换为value。

返回一个指向插入的项目的迭代器。

如果值是QJsonValue::Undefined，它将导致键被从对象中删除。然后，返回的迭代器将指向end()。

另请参阅 remove (), take (), QJsonObject::iterator ，和 end ().

QJsonValueRef QJsonObject:: operator[] (const & key )#

返回一个对key的值的引用。如果对象中没有键key的值，就会用一个QJsonValue::Null 的值创建一个，然后返回。

返回值是QJsonValueRef的类型，是QJsonArray和QJsonObject的辅助类。当你得到一个QJsonValueRef类型的对象时，你可以像使用一个QJsonValue的引用一样使用它。如果你对它进行赋值，赋值将适用于QJsonArray或QJsonObject中的元素，你从该元素中得到了引用。

另请参阅 value ().
