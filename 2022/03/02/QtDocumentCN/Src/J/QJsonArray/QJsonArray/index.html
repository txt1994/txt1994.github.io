详细描述#

一个JSON数组是一个值的列表。这个列表可以通过从数组中插入和删除QJsonValue的方式来进行操作。

QJsonArray可以转换为QVariantList，也可以从QVariantList转换。你可以用size()、insert()和removeAt()查询其中的条目数，并使用标准的C++迭代器模式对其内容进行迭代。

QJsonArray是一个隐式共享的类，只要它没有被修改，就会与它所创建的文档共享数据。

你可以通过QJsonDocument将数组转换为基于文本的JSON。

另请参阅 在 Qt 中支持 JSON and JSON 保存游戏范例 .

void QJsonArray:: removeAt ( qsizetype i )#

void QJsonArray:: removeFirst ()#

删除数组中的第一个项目。调用此函数等同于调用removeAt(0)。数组不能是空的。如果数组可以为空，在调用此函数之前，请调用isEmpty()。

另请参阅 removeAt () 和 removeLast ().

void QJsonArray:: removeLast ()#

删除数组中的最后一个项目。调用此函数等同于调用removeAt(size() - 1) 。数组不能是空的。如果数组可以为空，在调用此函数之前，请调用isEmpty()。

另请参阅 removeAt () 和 removeFirst ().

QJsonValueRef QJsonArray:: operator[] ( qsizetype i )#

返回索引位置i的值，作为一个可修改的引用。i必须是数组中有效的索引位置（即：0 <= i < size() ）。

返回值是QJsonValueRef的类型，是QJsonArray和QJsonObject的辅助类。当你得到一个QJsonValueRef类型的对象时，你可以像使用一个QJsonValue的引用那样使用它。如果你对它进行赋值，赋值将适用于QJsonArray的QJsonObject中的字符，而你正是从那里得到了这个引用。

另请参阅 at ().
