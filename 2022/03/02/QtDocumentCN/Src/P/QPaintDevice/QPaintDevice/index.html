详细描述#

绘画设备是一个二维空间的抽象，可以使用QPainter进行绘画。它的默认坐标系统的原点位于左上角的位置。X向右增加，Y向下增加。单位是一个像素。

QPaintDevice的绘图功能目前由QWidget、QImage、QPixmap、QPicture和QPrinter 子类实现。

要实现对一个新的后端的支持，你必须派生自QPaintDevice，并重新实现虚拟的paintEngine()函数来告诉QPainter应该使用哪个绘画引擎来在这个特定的设备上绘画。请注意，你也必须创建一个相应的绘画引擎，以便能够在设备上绘画，即从QPaintEngine派生出来并重新实现其虚拟函数。

警告： Qt要求在创建任何绘画设备之前存在一个QGuiApplication 对象。绘画设备会访问窗口系统资源，而这些资源在应用程序对象被创建之前不会被初始化。

QPaintDevice类提供了几个返回各种设备指标的函数。depth()函数返回其位深度（位平面的数量）。height](#height)()函数以默认的坐标系单位返回其高度（例如，QPixmap和QWidget的像素），而heightMM()返回设备的高度（毫米）。同样，width()和widthMM()函数分别以默认坐标系单位和毫米为单位返回设备的宽度。另外，受保护的metric()函数可以通过指定所需的PaintDeviceMetric作为参数来获取公制信息。

logicalDpiX()和logicalDpiY()函数返回设备的水平和垂直分辨率，单位为每英寸点数。physicalDpiX()和physicalDpiY()函数也返回设备的分辨率，单位是点/英寸，但要注意，如果逻辑和物理分辨率不同，相应的QPaintEngine必须处理这个映射关系。最后，colorCount()函数返回绘画设备可用的不同颜色的数量。

See also QPaintEngine, QPainter, Coordinate System, and Paint System.

成员类型文档#

enum QPaintDevice::PaintDeviceMetric#

描述了涂装设备的各种指标。

Constant Value Description
QPaintDevice::PdmWidth 1 绘画设备的宽度，以默认的坐标系单位（例如：QPixmap和QWidget的像素）。也请参见width()。
QPaintDevice::PdmHeight 2 绘画设备的高度，以默认的坐标系单位（例如，QPixmap和QWidget的像素）。也请参见height()。
QPaintDevice::PdmWidthMM 3 绘画设备的宽度，单位是毫米。参见widthMM()。
QPaintDevice::PdmHeightMM 4 涂装设备的高度，单位是毫米。参见heightMM()。
QPaintDevice::PdmNumColors 5 涂装设备可用的不同颜色的数量。参见colorCount()。
QPaintDevice::PdmDepth 6 绘图设备的位深度（位平面的数量）。参见depth()。
QPaintDevice::PdmDpiX 7 设备的水平分辨率，单位是每英寸点数。参见logicalDpiX()。
QPaintDevice::PdmDpiY 8 设备的垂直分辨率，单位为每英寸点数。另请参阅logicalDpiY()。
QPaintDevice::PdmPhysicalDpiX 9 设备的水平分辨率，单位为每英寸点数。另请参阅physicalDpiX()。
QPaintDevice::PdmPhysicalDpiY 10 设备的垂直分辨率，单位为每英寸点数。参见physicalDpiY()。
QPaintDevice::PdmDevicePixelRatio 11 设备的设备像素比。常见的值是普通dpi显示器为1，高dpi “视网膜 “显示器为2。
QPaintDevice::PdmDevicePixelRatioScaled 12 设备的缩放设备像素比。这与PdmDevicePixelRatio相同，只是该值被一个恒定的因子缩放，以支持具有分数比例因子的绘画设备。使用的恒定缩放系数是devicePixelRatioFScale()。这个枚举值已在Qt 5.6中引入。

See also metric() and devicePixelRatio().

成员函数文档#

[protected]QPaintDevice::QPaintDevice()#

构建一个绘画设备。这个构造函数只能从QPaintDevice的子类中调用。

[virtual]QPaintDevice::~QPaintDevice()#

销毁绘画设备并释放窗口系统资源。

int QPaintDevice::colorCount() const#

返回该绘画设备可用的不同颜色的数量。如果可用的颜色数量太多，无法用int数据类型表示，那么将返回INT_MAX。

int QPaintDevice::depth() const#

返回绘画设备的位深度（位平面的数量）。

QPaintDevice::devicePixelRatio() const#

返回设备的像素比。

常见的值是普通dpi显示器为1，高dpi “retina “显示器为2。

[since 5.6] QPaintDevice::devicePixelRatioF() const#

返回设备的设备像素比为一个浮点数。

这个函数在Qt 5.6中引入。

int QPaintDevice::height() const#

以默认的坐标系单位返回绘制设备的高度（例如，对于 QPixmap 和 QWidget 的像素）。

See also heightMM().

int QPaintDevice::heightMM() const#

返回绘画设备的高度，单位是毫米。由于平台的限制，可能无法使用这个函数来确定屏幕上小部件的实际物理尺寸。

See also height().

int QPaintDevice::logicalDpiX() const#

返回设备的水平分辨率，单位是点/英寸，这在计算字体大小时使用。对于X11来说，这通常与从widthMM()计算出来的结果相同。

注意，如果logicalDpiX()不等于physicalDpiX()，相应的QPaintEngine必须处理分辨率的映射。

See also logicalDpiY() and physicalDpiX().

int QPaintDevice::logicalDpiY() const#

返回设备的垂直分辨率，单位是每英寸点数，在计算字体大小时使用。对于X11来说，这通常与从heightMM()计算出来的结果相同。

注意，如果logicalDpiY()不等于physicalDpiY()，相应的QPaintEngine必须处理这个分辨率映射。

See also logicalDpiX() and physicalDpiY().

[virtual protected]int QPaintDevice::metric( metric) const#

返回给定绘画设备的度量信息 metric.

See also PaintDeviceMetric.

[pure virtual] *QPaintDevice::paintEngine() const#

返回一个指向用于在设备上绘图的绘图引擎的指针。

bool QPaintDevice::paintingActive() const#

如果该设备目前正在被绘制，即有人调用了QPainter::begin()但还没有调用QPainter::end()，则返回true；否则返回false。

See also QPainter::isActive().

int QPaintDevice::physicalDpiX() const#

返回设备的水平分辨率，单位是点/英寸。例如，在打印时，这个分辨率是指物理打印机的分辨率。另一方面，逻辑DPI是指实际绘画引擎使用的分辨率。

注意，如果physicalDpiX()不等于logicalDpiX()，相应的QPaintEngine必须处理这个分辨率映射。

See also physicalDpiY() and logicalDpiX().

int QPaintDevice::physicalDpiY() const#

返回设备的水平分辨率，单位是点/英寸。例如，在打印时，这个分辨率是指物理打印机的分辨率。另一方面，逻辑DPI是指实际绘画引擎所使用的分辨率。

注意，如果physicalDpiY()不等于logicalDpiY()，相应的QPaintEngine必须处理这个分辨率映射。

See also physicalDpiX() and logicalDpiY().

int QPaintDevice::width() const#

以默认的坐标系单位返回绘制设备的宽度（例如，对于QPixmap和QWidget的像素）。

See also widthMM().

int QPaintDevice::widthMM() const#

返回绘画设备的宽度，单位是毫米。由于平台的限制，可能无法使用这个函数来确定屏幕上一个小部件的实际物理尺寸。

See also width().
