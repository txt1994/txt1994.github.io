详细描述#

一支笔有style()、width()、brush()、capStyle()和joinStyle() 。

笔的样式定义了线的类型。笔刷用于填充用笔生成的笔触。使用 QBrush 类来指定填充样式。笔帽样式决定了可以使用QPainter绘制的线端帽，而连接样式描述了如何绘制两条线之间的连接。笔宽可以用整数（width()）和浮点（widthF()）的精度来指定。线条宽度为0，表示是化妆笔。这意味着笔宽总是画成一个像素宽，与画师上设置的变换无关。

使用相应的setStyle()、setWidth()、setBrush()、setCapStyle()和setJoinStyle()函数，可以轻松修改各种设置（注意，在改变笔的属性时必须重置画师的笔）。

例子:

QPainter painter(this);
QPen pen(Qt::green, 3, Qt::DashDotLine, Qt::RoundCap, Qt::RoundJoin);
painter.setPen(pen);

这相当于：

QPainter painter(this);
QPen pen;  // creates a default pen

pen.setStyle(Qt::DashDotLine);
pen.setWidth(3);
pen.setBrush(Qt::green);
pen.setCapStyle(Qt::RoundCap);
pen.setJoinStyle(Qt::RoundJoin);

painter.setPen(pen);

默认的笔是具有1个宽度的纯黑色笔刷，方形笔帽样式（Qt::SquareCap），以及斜面连接样式（Qt::BevelJoin）。

此外，QPen还提供了color()和setColor()便利函数，分别用于提取和设置笔刷的颜色。钢笔也可以被比较和流化。

关于一般绘画的更多信息，请参阅Paint System文档。

pen 样式#

Qt 提供了几种内置的样式，由 Qt::PenStyle 枚举表示：
Qt::SolidLine Qt::DashLine Qt::DotLine
Qt::DashDotLine Qt::DashDotDotLine Qt::CustomDashLine

只需使用 setStyle() 函数将笔样式转换为任何内置样式，除了我们将回来的 Qt::CustomDashLine 样式 到不久。 将样式设置为 Qt::NoPen 告诉画家不要画线或轮廓。 默认的画笔样式是 Qt::SolidLine。

从 Qt 4.1 开始，还可以使用 setDashPattern() 函数指定自定义破折号图案，该函数将笔的样式隐式转换为 Qt::CustomDashLine。 模式参数 QList 必须指定为偶数个 qreal 条目，其中条目 1、3、5… 是破折号和 2, 4, 6… 是空格。 例如，上面显示的自定义模式是使用以下代码创建的：

QPen pen;
QList<qreal> dashes;
qreal space = 4;

dashes << 1 << space << 3 << space << 9 << space
    << 27 << space << 9 << space;

pen.setDashPattern(dashes);

请注意，破折号图案是以笔的宽度为单位来指定的，例如，一个长度为5、宽度为10的破折号是50像素长。

当前设置的破折号图案可以用dashPattern()函数来检索。使用isSolid()函数来确定笔是否有实体填充。

cap 样式#

帽子样式定义了如何使用QPainter绘制线条的端点。帽子样式只适用于宽线，即宽度为1或更大时。Qt::PenCapStyle 枚举提供了以下样式：
Qt::SquareCap Qt::FlatCap Qt::RoundCap

Qt::SquareCap 样式是一个覆盖端点的方形线端，并超出其一半的线宽。Qt::FlatCap 样式是一个不覆盖线的端点的方形线端。而 Qt::RoundCap 样式是一个覆盖端点的圆形线端。

默认是 Qt::SquareCap。

当笔宽为0或1时，是否绘制端点取决于笔帽样式。使用 Qt::SquareCap 或 Qt::RoundCap 会绘制，使用 Qt::FlatCap 则不绘制。

Join 样式#

连接样式定义了如何使用QPainter绘制两条连接线之间的连接。连接样式只适用于宽线，即宽度为1或更大时。Qt::PenJoinStyle 枚举提供了以下样式：
Qt::BevelJoin Qt::MiterJoin Qt::RoundJoin

Qt::BevelJoin 样式填充了两条线之间的三角形缺口。Qt::MiterJoin 样式将两条线延伸到一个角度处相遇。而 Qt::RoundJoin 样式在两条线之间填充了一个圆弧。

默认是 Qt::BevelJoin.

当 Qt::MiterJoin 样式被应用时，可以使用 setMiterLimit() 函数来指定米特尔连接可以从连接点延伸多远。miterLimit()用于减少线条接近平行的线条连接处的伪影。

miterLimit()必须以笔的宽度为单位指定，例如，宽度为10的miter limit为5，长度为50像素。默认的米特限制是2，也就是笔宽的两倍（像素）。

**The Path Stroking Example**Path Stroking示例展示了Qt的内置破折号图案，并展示了如何使用自定义图案来扩展可用图案的范围。

See also QPainter, QBrush, Path Stroking Example, and Scribble Example.

bool QPen::isCosmetic() const#

如果该笔是化妆品，返回 “true”；否则返回 “false”。

化妆笔用于绘制具有恒定宽度的笔画，而不考虑应用于它们的QPainter的任何变换。用化妆笔画一个形状可以确保它的轮廓在不同的比例因子下有相同的厚度。

默认情况下，零宽度的笔是美容的。

See also setCosmetic() and widthF().

See also style() and dashPattern().

void QPen::setDashOffset( offset)#

将此笔的破折号偏移（破折号图案的起始点）设置为offset指定的值。该偏移量是以用于指定破折号图案的单位来衡量的。

例如，一个每个笔画长4个单位，后面有2个单位的间隙的图案，当作为线画时，将从笔画开始。然而，如果破折号偏移被设置为4.0，任何线画将从间隙开始。4.0以下的偏移值将导致先画出部分笔画，4.0到6.0之间的偏移值将导致线条从部分间隙开始。

注意：这隐含地将笔的样式转换为Qt::CustomDashLine。

See also dashOffset().

void QPen::setDashPattern(const <> &pattern)#

将此笔的破折号图案设置为给定的pattern。这隐含地将笔的样式转换为Qt::CustomDashLine。

该模式必须被指定为偶数的正数条目，其中条目1、3、5…是破折号，2、4、6…是空格。比如说：

QPen pen; 
QList dashes; 
qreal space = 4; 
dashes << 1 << space << 3 << space << 9 << space << 27 << space << 9 << space; pen.setDashPattern(dashes);

破折号图案是以笔的宽度为单位来指定的；例如，一个长度为5、宽度为10的破折号是50像素长。注意，宽度为零的笔相当于宽度为1像素的化妆笔。

每个破折号也会受到笔帽样式的影响，所以一个1的破折号在设置了方形笔帽后会在每个方向延伸0.5像素，导致总宽度为2。

请注意，默认的笔帽样式是Qt::SquareCap，这意味着一个正方形的线端覆盖了端点，并在它之外延伸了一半的线宽。

See also setStyle(), dashPattern(), setCapStyle(), and setCosmetic().

void QPen::setJoinStyle( style)#

将笔的连接样式设置为给定的style。默认值是Qt::BevelJoin。

See also joinStyle() and Join Style.

void QPen::setMiterLimit( limit)#

将此笔的割裂极限设置为给定的limit。

铣削极限描述了铣削连接可以从连接点延伸多远。这用于减少线与线之间接近平行时的假象。

这个值只有在钢笔样式被设置为 Qt::MiterJoin 时才有效果。这个值是以笔的宽度为单位指定的，例如，在宽度为10的情况下，miter limit为5，就是50像素长。默认的割裂限制是2，也就是笔的宽度的两倍（像素）。

See also miterLimit(), setJoinStyle(), and Join Style.

void QPen::setStyle( style)#

将笔的样式设置为给定的style。

请参阅 Qt::PenStyle 文档以了解可用的风格列表。从Qt 4.1开始，也可以使用setDashPattern()函数指定一个自定义的破折号样式，该函数隐含地将笔的样式转换为Qt::CustomDashLine。

注意：这个函数将破折号的偏移量重置为零。

See also style() and Pen Style.

void QPen::setWidth(int width)#

将笔的宽度设置为整数精度的给定width像素。

行的宽度为零表示是化妆笔。这意味着笔宽总是被画成一个像素宽，与画师上设置的变换无关。

不支持设置一个负值的笔宽。

See also setWidthF() and width().

void QPen::setWidthF( width)#

将笔宽设置为给定的width，以浮点精度计算。

线条宽度为零表示是化妆笔。这意味着笔宽总是被画成一个像素宽，与画师的变换无关。

不支持设置一个负值的笔宽。

See also setWidth() and widthF().

QPen::style() const#

返回笔的样式。

See also setStyle() and Pen Style.

void QPen::swap( &other)#

用这支笔交换other笔。这个操作是非常快的，而且永远不会失败。

int QPen::width() const#

以整数精度返回笔的宽度。

See also setWidth() and widthF().

QPen::widthF() const#

以浮点精度返回笔的宽度。

See also setWidthF() and width().

QPen::operator QVariant() const#

将笔作为QVariant返回。

bool QPen::operator!=(const &pen) const#

如果笔与给定的pen不同，则返回 “true”；否则返回 “false”。如果两支笔有不同的风格、宽度或颜色，就是不同的。

See also operator==().

bool QPen::operator==(const &pen) const#

如果笔与给定的pen相等，则返回true；否则返回false。如果两支笔的样式、宽度和颜色相等，则为相等。

See also operator!=().
