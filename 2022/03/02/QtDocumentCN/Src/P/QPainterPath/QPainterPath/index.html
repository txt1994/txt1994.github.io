详细描述#

画师路径是一个由若干图形构件组成的对象，如矩形、椭圆、直线和曲线。构件可以在封闭的子路径中被连接起来，例如，作为一个矩形或一个椭圆。一个封闭的路径有重合的起点和终点。或者它们可以作为非封闭的子路径独立存在，如直线和曲线。

QPainterPath对象可以用于填充、勾勒和剪裁。要为一个给定的画师路径生成可填充的轮廓，请使用QPainterPathStroker类。与普通的绘图操作相比，绘图器路径的主要优势在于，复杂的形状只需要被创建一次；然后只需调用QPainter::drawPath()函数就可以多次绘制。

QPainterPath提供了一个函数集合，可以用来获取路径及其元素的信息。此外，还可以使用toReversed()函数来反转元素的顺序。还有几个函数可以将这个绘画者路径对象转换为多边形表示。

构成一个QPainterPath#

QPainterPath对象可以被构造成一个空的路径，有一个给定的起始点，或者是另一个QPainterPath对象的副本。创建后，可以使用lineTo()、arcTo()、cubicTo()和quadTo()函数在路径上添加线条和曲线。线条和曲线从currentPosition()延伸到作为参数传递的位置。

QPainterPath对象的currentPosition()总是被添加的最后一个子路径的结束位置（或初始起始点）。使用moveTo()函数来移动currentPosition()，而不添加一个组件。moveTo()函数隐含地启动一个新的子路径，并关闭前一个子路径。另一种启动新子路径的方法是调用closeSubpath()函数，通过从currentPosition()添加一行到路径的起始位置来关闭当前路径。请注意，新路径的初始currentPosition()将是(0, 0)。

QPainterPath类还提供了几个方便的函数，可以将封闭的子路径添加到画师路径中。addEllipse(), addPath(), addRect(), addRegion() 和 addText()。addPolygon()函数添加了一个不封闭的子路径。事实上，这些函数都是moveTo()、lineTo()和cubicTo() 操作的集合。

此外，还可以用connectPath()函数将一个路径添加到当前路径中。但要注意的是，这个函数将通过添加一条线将当前路径的最后一个元素与给定路径的第一个元素连接起来。

下面是一个代码片段，展示了如何使用QPainterPath对象。

QPainterPath path; 
path.addRect(20, 20, 60, 60); 
path.moveTo(0, 0); 
path.cubicTo(99, 0,  50, 50,  99, 99); 
path.cubicTo(0, 99,  50, 50,  0, 0);
QPainter painter(this); 
painter.fillRect(0, 0, 100, 100, Qt::white); 
painter.setPen(QPen(QColor(79, 106, 25), 1, Qt::SolidLine,                    Qt::FlatCap, Qt::MiterJoin)); 
painter.setBrush(QColor(122, 163, 39));
painter.drawPath(path);

画家的路径在构建时最初是空的。我们首先添加一个矩形，它是一个封闭的子路径。然后我们添加两条贝塞尔曲线，它们共同构成了一个封闭的子路径，尽管它们并不是单独封闭的。最后，我们画出整个路径。路径是用默认的填充规则Qt::OddEvenFill来填充的。Qt提供了两种填充路径的方法。

Qt::OddEvenFill Qt::WindingFill

请参阅 Qt::FillRule 文档以了解规则的定义。画家路径当前设置的填充规则可以通过fillRule()函数检索，并通过setFillRule()函数改变。

QPainterPath信息#

QPainterPath类提供了一个函数集合，用于返回有关路径及其元素的信息。

currentPosition()函数返回最后添加的子路径的端点（或初始起始点）。elementAt()函数可以用来检索各种子路径元素，元素的*数量可以用elementCount()函数检索，isEmpty()函数告诉这个QPainterPath对象是否包含任何元素。

controlPointRect()函数返回包含这个路径中所有点和控制点的矩形。这个函数的计算速度明显快于精确的boundingRect()，后者以浮点精度返回这个画师路径的边界矩形。

最后，QPainterPath提供了contains()函数，可以用来确定一个给定的点或矩形是否在路径内，以及intersects()函数，确定一个给定的矩形内的任何点是否也在这个路径内。

QPainterPath转换#

由于兼容性的原因，可能需要简化绘画者路径的表示。QPainterPath提供了toFillPolygon()、toFillPolygons()和toSubpathPolygons()函数，将绘画者路径转换成多边形。toFillPolygon()将绘画者的路径作为一个单一的多边形返回，而后两个函数则返回一个多边形的列表。

提供toFillPolygons()和toSubpathPolygons()函数是因为绘制几个小的多边形通常比绘制一个大的多边形要快，尽管绘制的总点数是一样的。两者之间的区别在于它们返回的多边形的*数量。toSubpathPolygons()为每个子路径创建一个多边形，而不考虑相交的子路径（即重叠的边界矩形），而toFillPolygons()函数对重叠的子路径只创建一个多边形。

toFillPolygon()和toFillPolygons()函数首先将所有子路径转换为多边形，然后使用回卷技术来确保重叠的子路径可以使用正确的填充规则进行填充。请注意，倒卷在多边形中插入了额外的线条，所以填充多边形的轮廓与路径的轮廓不一致。

成员类型文档#

enum QPainterPath::ElementType#

这个枚举描述了用于连接子路径中顶点的元素类型。

请注意，使用addEllipse()、addPath()、addPolygon()、addRect()作为封闭子路径加入的元素。addRegion()和addText()等便利函数，实际上是使用moveTo()、lineTo()和cubicTo()函数作为独立元素的集合添加到路径。

Constant Value Description
QPainterPath::MoveToElement 0 一个新的子路径。另见moveTo()。
QPainterPath::LineToElement 1 一条线。也见lineTo()。
QPainterPath::CurveToElement 2 一条曲线。参见cubicTo()和quadTo()。
QPainterPath::CurveToDataElement 3 在CurveToElement元素中描述一条曲线所需的额外数据。

See also elementAt() and elementCount().

void QPainterPath::addEllipse(const &boundingRectangle)#

在指定的boundingRectangle内创建一个椭圆并将其作为封闭子路径添加到画家路径。

椭圆由一条顺时针曲线组成，从零度开始和结束（3 点钟位置）。

QLinearGradient myGradient; 
QPen myPen; 
QRectF boundingRectangle; 
QPainterPath myPath; 
myPath.addEllipse(boundingRectangle); 
QPainter painter(this); 
painter.setBrush(myGradient); 
painter.setPen(myPen); 
painter.drawPath(myPath);

See also arcTo(), QPainter::drawEllipse(), and Composing a QPainterPath.

QPainterPath::angleAtPercent( t) const#

返回在百分比t处的路径切线的角度。参数t必须在0到1之间。

角度的正值意味着逆时针方向，而负值意味着顺时针方向。零度是在3点钟的位置。

请注意，与其他百分比方法类似，如果路径中存在曲线，百分比测量与长度的关系不是线性的。当曲线存在时，百分比参数被映射到贝塞尔方程的t参数上。

void QPainterPath::arcMoveTo(const &rectangle, angle)#

创建一个位于弧线上的移动，该弧线以angle占据给定的rectangle。

角度的单位是度。顺时针方向的弧线可以用负的角度来指定。

See also moveTo() and arcTo().

void QPainterPath::arcMoveTo( x, y, width, height, angle)#

这是一个重载函数。

创建一个位于弧线上的移动到，该弧线在angle处占据QRectF(x, y, width, height)。

void QPainterPath::arcTo(const &rectangle, startAngle, sweepLength)#

创建一个占据给定rectangle的弧，从指定的startAngle开始，逆时针延伸sweepLength度。

角度的单位是度。顺时针方向的弧可以用负的角度来指定。

请注意，如果弧的起点和当前位置还没有连接，这个函数会将它们连接起来。弧线被添加后，当前位置是弧线的最后一点。要画一条线回到第一个点，请使用closeSubpath()函数。

QPainterPath myPath; 
myPath.moveTo(center);
myPath.arcTo(boundingRect, startAngle,sweepLength);
QPainter painter(this); 
painter.setBrush(myGradient); 
painter.setPen(myPen); 
painter.drawPath(myPath);

See also arcMoveTo(), addEllipse(), QPainter::drawArc(), QPainter::drawPie(), and Composing a QPainterPath.

QPainterPath::toFillPolygon(const &matrix = QTransform()) const#

使用QTransform matrix将路径转换为一个多边形，并返回多边形。

创建多边形的方法是首先将所有子路径转换为多边形，然后使用回卷技术，确保重叠的子路径可以使用正确的填充规则进行填充。

注意，倒卷在多边形中插入了加线，因此填充多边形的轮廓与路径的轮廓不一致。

See also toSubpathPolygons(), toFillPolygons(), and QPainterPath Conversion.

<> QPainterPath::toFillPolygons(const &matrix = QTransform()) const#

使用QTransform matrix将路径转换为多边形的列表，并返回该列表。

该函数与toFillPolygon()函数的不同之处在于，它创建了几个多边形。之所以提供这个函数，是因为绘制几个小的多边形通常比绘制一个大的多边形要快，尽管绘制的总点数是一样的。

toFillPolygons()函数与toSubpathPolygons()函数不同的是，它只为有重叠边界矩形的子路径创建多边形。

像toFillPolygon()函数一样，这个函数使用了一个回卷技术，以确保重叠的子路径可以使用正确的填充规则进行填充。注意，倒卷在多边形中插入了加线，所以填充多边形的轮廓与路径的轮廓不一致。

See also toSubpathPolygons(), toFillPolygon(), and QPainterPath Conversion.

QPainterPath::toReversed() const#

创建并返回路径的反转副本。

反转的是元素的顺序：如果一个QPainterPath是按照指定的顺序调用moveTo()、lineTo()和cubicTo()函数组成的，反转的副本是按照调用cubicTo()、lineTo()和moveTo()组成。

<> QPainterPath::toSubpathPolygons(const &matrix = QTransform()) const#

使用QTransform matrix将路径转换为多边形列表，并返回该列表。

这个函数为每个子路径创建一个多边形，而不考虑相交的子路径（即重叠的边界矩形）。为了确保这种重叠的子路径被正确填充，请使用toFillPolygons()函数代替。

See also toFillPolygons(), toFillPolygon(), and QPainterPath Conversion.
