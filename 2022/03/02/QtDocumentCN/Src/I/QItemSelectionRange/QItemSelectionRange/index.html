详细描述#

一个QItemSelectionRange包含了一个模型中选定项的范围的信息。一个项的范围是一个连续的模型项的数组，扩展到覆盖一些相邻的行和列，有一个共同的父项；这可以被看作是一个表格中的二维单元块。一个选择范围有一个top(), left 一个bottom(), right() 和一个parent() 。

QItemSelectionRange类是模型/视图类且属于Qt的模型/视图框架之一。

使用indexes()函数可以获得选择范围内包含的模型项目。使用QItemSelectionModel::selectedIndexes()来获取一个视图的所有选择项的列表。

你可以通过使用contains()函数来确定一个给定的模型项目是否位于一个特定的范围内。范围也可以使用平等和不平等的重载运算符进行比较，intersects()函数允许你确定两个范围是否重叠。

另请参阅 模型/视图编程 , QAbstractItemModel , QItemSelection ，和 QItemSelectionModel .

QItemSelectionRange:: QItemSelectionRange (const & index )#

构建一个新的选择范围，只包含由模型索引index指定的模型项目。

QItemSelectionRange:: QItemSelectionRange (const & topLeft , const & bottomRight )#

构建一个新的选择范围，只包含由topLeft和bottomRight指定的索引。

QItemSelectionRange:: QItemSelectionRange ()#

构建一个空的选择范围。

int QItemSelectionRange:: bottom () const#

返回选择范围中最下层的选择行所对应的行索引。

const &QItemSelectionRange:: bottomRight () const#

返回位于选择范围右下角的项目的索引。

另请参阅 bottom (), right ()，和 topLeft ().

bool QItemSelectionRange:: contains (const & index ) const#

返回 true 如果由index指定的模型项目位于所选项目的范围内，否则返回 false 。

bool QItemSelectionRange:: contains ( int row , int column , const & parentIndex ) const#

这是重载函数。

返回 true 如果由( row , column )指定的模型项目，并且以parentIndex作为父项目，位于所选项目的范围内；否则返回false。

int QItemSelectionRange:: height () const#

返回选择范围内被选中的行数。

QItemSelectionRange:: indexes () const#

返回存储在选择中的模型索引项目的列表。

QItemSelectionRange:: intersected (const & other ) const#

返回一个新的选择范围，只包含在选择范围和other 选择范围中都能找到的项目。

bool QItemSelectionRange:: intersects (const & other ) const#

返回 true 如果这个选择范围与所给的other 范围相交（重叠）；否则返回 false 。

bool QItemSelectionRange:: isEmpty () const#

返回 true 如果选择范围不包含任何项目，或者只有被禁用或被标记为不可选择的项目。

bool QItemSelectionRange:: isValid () const#

如果选择范围有效，返回 “true”；否则返回 “false”。

int QItemSelectionRange:: left () const#

返回选择范围中最左边的选择列所对应的列索引。

const *QItemSelectionRange:: model () const#

返回选择范围内的项目所属的模型。

QItemSelectionRange:: parent () const#

返回选择范围内的项目的父模型项目索引。

int QItemSelectionRange:: right () const#

返回选择范围中最右边的选择列所对应的列索引。

[since 5.6] void QItemSelectionRange:: swap ( & other )#

将此选择范围的内容与other 的内容交换。 。此函数非常快，且从不失败。

该函数在 Qt 5.6 引入。

int QItemSelectionRange:: top () const#

返回选择范围中最上层的选择行所对应的行索引。

const &QItemSelectionRange:: topLeft () const#

返回位于选择范围左上角的项目的索引。

另请参阅 top (), left ()，和 bottomRight ().

int QItemSelectionRange:: width () const#

返回选择范围内被选中的列的数量。

bool QItemSelectionRange:: operator!= (const & other ) const#

返回 true 如果选择范围与给定的other 范围不同；否则返回 false 。

bool QItemSelectionRange:: operator== (const & other ) const#

返回 true 如果选择的范围与给定的other 范围完全相同；否则返回 false 。
