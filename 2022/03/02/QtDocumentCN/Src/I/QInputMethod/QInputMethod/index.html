详细描述#

QInputMethod被文本编辑器用来集成到平台上的文本输入法，更常见的是被应用程序视图用来查询各种文本输入法相关的信息，如虚拟键盘的可见性和键盘尺寸。

Qt Quick还通过Qt全局对象在QML中提供对QInputMethod的访问，作为Qt.inputMethod特性。

成员类型文档编制#

enum QInputMethod:: 动作#

表示用户执行的行动的种类。

常量 值 描述
QInputMethod::Click 0 正常的点击/敲击
QInputMethod::ContextMenu 1 点击上下文菜单（如右键或点击并按住）。

另请参阅 invokeAction ().

[read-only] anchorRectangle : const #

在窗口坐标中输入项目的锚定矩形。

锚定矩形经常被各种文本编辑控件使用，如文本预测弹出窗口，用于跟踪文本选择。

[read-only] animating : const bool#

当虚拟键盘正在打开或关闭时为真。

当键盘完全打开或关闭时，动画效果为假。当animating是true和visibility是true键盘正在打开。当animating是true和visibility是false时，键盘被关闭。

在窗口坐标中输入项目的光标矩形。

光标矩形经常被各种文本编辑控件使用，如文本预测弹出窗口，用于跟踪正在输入的文本。

剪切的输入矩形通常被各种输入法用来确定有多少屏幕空间可供输入法使用（如虚拟键盘）。

成员函数文档编制#

[slot] void QInputMethod:: commit ()#

将用户目前正在写的字提交给编辑器。这个函数主要被具有文本预测功能的输入法所需要，也被那些用于输入字符的脚本与实际附加到编辑器的脚本不同的方法所需要。任何中断文字合成的行为都需要通过调用commit()函数来刷新合成状态，例如当光标被移到其他地方时。

[slot] void QInputMethod:: hide ()#

请求关闭虚拟键盘。

通常情况下，应用程序不需要调用这个函数，当文本编辑器失去焦点时，键盘应该自动关闭，例如当父视图关闭时。

[slot] void QInputMethod:: invokeAction ( a , int cursorPosition )#

当当前正在组成的单词被用户点击时，由输入项调用，如动作a和给定cursorPosition所示。输入法经常使用这一信息来向用户提供更多的单词建议。

[slot] void QInputMethod:: reset ()#

重置输入法的状态。例如，一个文本编辑器通常在插入一个文本之前调用这个方法，使小部件准备好接受一个文本。

当聚焦的编辑器改变时，输入法会自动重置。

[slot] void QInputMethod:: show ()#

要求打开虚拟键盘。如果平台不提供虚拟键盘，可见性保持为假。

通常情况下，应用程序不需要调用这个函数，键盘应该在文本编辑器获得焦点时自动打开。

[slot] void QInputMethod:: update ( queries )#

当编辑器的输入法查询属性出现状态变化时，由输入项调用，通知平台输入法。在调用该函数时，必须使用queries参数来告诉什么发生了变化，输入法可以用QInputMethodQueryEvent对它感兴趣的属性进行查询。

特别是每当光标位置发生变化时，调用更新是很重要的，因为这经常会导致其他查询属性，如周围的文本和文本选择也发生变化。为了方便起见，经常与光标位置一起变化的属性已经被分组到Qt::ImQueryInput值中。

[since 5.1] QInputMethod:: inputItemRectangle () const#

返回在输入项坐标中的输入项几何体。

该函数在 Qt 5.1 引入。

另请参阅 setInputItemRectangle ().

QInputMethod:: inputItemTransform () const#

返回从输入项目坐标到窗口坐标的转换。

另请参阅 setInputItemTransform ().

[static] QInputMethod:: queryFocusObject ( query , const & argument )#

发送 query 到当前聚焦对象采用参数 argument 并返回结果。

[since 5.1] void QInputMethod:: setInputItemRectangle (const & rect )#

设置输入项的几何形状为rect，以输入项的坐标为准。每当项目在场景中被移动或焦点被改变时，这需要由焦点窗口（如QQuickCanvas）更新。

该函数在 Qt 5.1 引入。

另请参阅 inputItemRectangle ().

void QInputMethod:: setInputItemTransform (const & transform )#

设置从输入项目坐标到窗口坐标的变换为transform。每当项目在场景中被移动时，项目的转换需要由焦点窗口（如QQuickCanvas）更新。

另请参阅 inputItemTransform ().

void QInputMethod:: setVisible ( bool visible )#

控制键盘的可见性。相当于调用 show () (若 visible is true ) or hide () (若 visible is false ).

另请参阅 isVisible (), show ()，和 hide ().
