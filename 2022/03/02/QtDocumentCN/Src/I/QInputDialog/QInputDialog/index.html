详细描述#

输入值可以是一个字符串、一个数字或一个列表中的一个项目。必须设置一个标签来告诉用户他们应该输入什么。

提供了五个静态便利函数。getText(), getMultiLineText(), getInt(), getDouble()，和getItem()。所有的函数都可以以类似的方式使用，例如:

bool ok;
QString text = QInputDialog::getText(this, tr("QInputDialog::getText()"),
                                     tr("User name:"), QLineEdit::Normal,
                                     QDir::home().dirName(), &ok);
if (ok && !text.isEmpty())
    textLabel->setText(text);

ok variable is set to true if the user clicks OK ; otherwise, it is set to false.

标准对话框例子展示了如何使用QInputDialog以及其他内置的Qt对话框。

另请参阅 QMessageBox and 标准对话框范例 .

enum QInputDialog:: InputDialogOption flags QInputDialog:: InputDialogOptions#

这个枚举指定了影响输入对话框外观的各种选项。

常量 值 描述
QInputDialog::NoButtons 0x00000001 不显示 OK 和 Cancel 按钮（对 “实时对话 “有用）。
QInputDialog::UseListViewForComboBoxItems 0x00000002 使用一个QListView而不是一个不可编辑的QComboBox来显示用setComboBoxItems()设置的项目。
QInputDialog::UsePlainTextEditForTextInput 0x00000004 使用QPlainTextEdit进行多行文本输入。这个值是在5.2中引入的。

InputDialogOptions类型是QFlags的一个类型定义。它存储了InputDialogOption值的一个OR组合。

另请参阅 options , setOption ()，和 testOption ().

enum QInputDialog:: InputMode#

这个枚举描述了可以为对话框选择的不同输入模式。

常量 值 描述
QInputDialog::TextInput 0 用来输入文本字符串。
QInputDialog::IntInput 1 用来输入整数。
QInputDialog::DoubleInput 2 用来输入具有双精度的浮点数字。

另请参阅 inputMode .

[static] double QInputDialog:: getDouble ( * parent , const & title , const & label , double value = 0, double min = -2147483647, double max = 2147483647, int decimals = 1, bool * ok = nullptr, flags = Qt::WindowFlags(), double step = 1)#

静态便利函数，从用户那里获得一个浮点数。

title是显示在对话框标题栏中的文本。label是显示给用户的文本（它应该说明应该输入什么）。value是默认的浮点数，行编辑将被设置为该数字。min和max是用户可以选择的最小值和最大值。decimals是数字可能有的最大小数位数。step是指用户按箭头按钮增加或减少数值时，数值的变化量。

如果ok为非空，ok将被设置为真，如果用户按了OK，则为假，如果用户按了Cancel。对话框的父级是parent 。该对话框将是模态的，并使用部件flags 。

这个函数返回用户已经输入的浮点数。

像这样使用这个静态函数:

bool ok;
double d = QInputDialog::getDouble(this, tr("QInputDialog::getDouble()"),
                                   tr("Amount:"), 37.56, -10000, 10000, 2, &ok,
                                   Qt::WindowFlags(), 1);
if (ok)
    doubleLabel->setText(QString("$%1").arg(d));

另请参阅 getText (), getInt (), getItem ()，和 getMultiLineText ().

[static] int QInputDialog:: getInt ( * parent , const & title , const & label , int value = 0, int min = -2147483647, int max = 2147483647, int step = 1, bool * ok = nullptr, flags = Qt::WindowFlags())#

静态便利函数，用于从用户那里获得一个整数输入。

title是显示在对话框标题栏中的文本。label是显示给用户的文本（它应该说明应该输入什么）。value是默认的整数，旋钮箱将被设置为该值。min和max是用户可以选择的最小值和最大值。step是用户按箭头按钮递增或递减数值时，数值的变化量。

若ok为非空，则ok将被设置为真，如果用户按了OK，则为假，如果用户按了Cancel。对话框的父级是parent 。对话框将是模态的，并使用部件的flags。

成功时，这个函数返回用户输入的整数；失败时，它返回初始值。

像这样使用这个静态函数:

bool ok;
int i = QInputDialog::getInt(this, tr("QInputDialog::getInt()"),
                             tr("Percentage:"), 25, 0, 100, 1, &ok);
if (ok)
    integerLabel->setText(tr("%1%").arg(i));

[static] QInputDialog:: getItem ( * parent , const & title , const & label , const & items , int current = 0, bool editable = true, bool * ok = nullptr, flags = Qt::WindowFlags(), inputMethodHints = Qt::ImhNone)#

静态便利函数，让用户从一个字符串列表中选择一个项目。

title是显示在对话框标题栏中的文本。label是显示给用户的文本（它应该说明应该输入什么）。items是插入到组合框中的字符串列表。current是项目的编号，应该是当前项目。inputMethodHints 是输入法提示，如果组合框是可编辑的，并且输入法是活动的，就会使用这个提示。

若 editable 为真，用户可以输入自己的文本；否则，用户只能选择现有的项目之一。

若ok为非空，如果用户按了OK，ok将被设置为真，如果用户按了Cancel，则为假。对话框的父级是parent。该对话框将是模态的，并使用部件flags* 。

这个函数返回当前项目的文本，如果editable为真，则返回组合框的当前文本。

像这样使用这个静态函数:

QStringList items;
items << tr("Spring") << tr("Summer") << tr("Fall") << tr("Winter");
bool ok;
QString item = QInputDialog::getItem(this, tr("QInputDialog::getItem()"),
                                     tr("Season:"), items, 0, false, &ok);
if (ok && !item.isEmpty())
    itemLabel->setText(item);

另请参阅 getText (), getInt (), getDouble ()，和 getMultiLineText ().

[static, since 5.2] QInputDialog:: getMultiLineText ( * parent , const & title , const & label , const & text = QString(), bool * ok = nullptr, flags = Qt::WindowFlags(), inputMethodHints = Qt::ImhNone)#

静态便利函数，从用户那里获得一个多行字符串。

title是显示在对话框标题栏中的文本。label是显示给用户的文本（它应该说明应该输入什么）。text是放在纯文本编辑中的默认文本。inputMethodHints 是输入法提示，如果输入法被激活，将在编辑部件中使用。

若ok为非空，ok将被设置为真，如果用户按了OK，则为假，如果用户按了Cancel。对话框的父级是parent 。该对话框将是模态的，并使用指定的部件flags 。

如果对话框被接受，这个函数返回对话框的纯文本编辑中的文本。如果对话框被拒绝，一个空的QString被返回。

像这样使用这个静态函数:

bool ok;
QString text = QInputDialog::getMultiLineText(this, tr("QInputDialog::getMultiLineText()"),
                                              tr("Address:"), "John Doe\nFreedom Street", &ok);
if (ok && !text.isEmpty())
    multiLineTextLabel->setText(text);

该函数在 Qt 5.2 引入。

  • textValueSelected() 如果member的第一个参数是QString。
  • intValueSelected() 如果成员的第一个参数是一个int。
  • doubleValueSelected()如果成员的第一个参数是一个双数。
  • accepted() 如果member没有参数。

信号将断开槽连接，当对话框被关闭时。
