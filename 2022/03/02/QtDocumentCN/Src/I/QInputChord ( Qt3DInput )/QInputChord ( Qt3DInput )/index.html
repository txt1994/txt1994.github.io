# QInputChord Class

class Qt3DInput ::QInputChord

QInputChord表示一组QAbstractActionInput的，必须一次性触发。 更多…

Header: #include
CMake: find_package(Qt6 COMPONENTS 3dinput REQUIRED) target_link_libraries(mytarget PRIVATE Qt6::3dinput)
qmake: QT += 3dinput
Since: Qt 5.7
实例化： InputChord
继承： Qt3DInput::QAbstractActionInput

特性文档编制#

timeout : int#

输入和弦中所有QAbstractActionInput的时间必须在此范围内被触发。时间的单位是毫秒。

访问函数：

int timeout () const
void setTimeout (int timeout )

通知程序信号：
