详细描述#

QImageCapture类是一个高水平的图像记录类。它不是用来单独使用的，而是用来访问其他媒体对象的媒体记录功能，比如QCamera。

QMediaCaptureSession captureSession;
camera = new QCamera;
captureSession.setCamera(camera);
QVideoWidget *preview = new QVideoWidget();
preview->show();
captureSession.setVideoOutput(preview);
imageCapture = new QImageCapture(camera);
captureSession.setImageCapture(imageCapture);
camera->start();
//on shutter button pressed
imageCapture->capture();

另请参阅 QCamera .

enum QImageCapture:: Error#

常量 值 描述
QImageCapture::NoError 0 没有错误。
QImageCapture::NotReadyError 1 该服务还没有准备好用于捕获。
QImageCapture::ResourceError 2 设备未准备好或无法使用。
QImageCapture::OutOfSpaceError 3 设备上没有剩余空间。
QImageCapture::NotSupportedFeatureError 4 设备不支持静态图像捕捉。
QImageCapture::FormatError 5 目前的格式不被支持。

成员函数文档编制#

QImageCapture:: QImageCapture ( * parent = nullptr)#

从一个parent构造一个图像捕捉对象，它可以捕捉由相机产生的单个静态图像。

你必须将一个图像捕捉对象和一个QCamera连接到一个捕捉会话以捕捉图像。

[slot] int QImageCapture:: capture ()#

捕获图像并将其作为一个QImage来使用。这个操作在大多数情况下是异步的，随后是信号 QImageCapture::imageExposed (), QImageCapture::imageCaptured () 或 QImageCapture::error ()。

QImageCapture::capture返回捕获Id参数，与imageExposed(), imageCaptured() 和 imageSaved() 信号一起使用。

另请参阅 isReadyForCapture ().

[slot] int QImageCapture:: captureToFile (const & file = QString())#

捕获图像并将其保存到file。这个操作在大多数情况下是异步的，后面有信号 QImageCapture::imageExposed (), QImageCapture::imageCaptured (), QImageCapture::imageSaved () 或 QImageCapture::error () 。

如果传递了一个空的*文件，相机后端会选择系统中照片的默认位置和命名方案，如果只指定了文件名而没有指定完整的路径，图像将被保存到默认的目录中，完整的路径用 imageCaptured () 和 imageSaved () 信号报告。

QCamera保存了所有的捕捉参数，如曝光设置或图像处理参数，所以在调用capture()后，对相机参数的改变不会影响之前的捕捉请求。

QImageCapture::capture返回捕获Id参数，与imageExposed()、imageCaptured()和imageSaved() 信号一起使用。

另请参阅 isReadyForCapture ().

[signal] void QImageCapture:: imageSaved ( int id , const & fileName )#
