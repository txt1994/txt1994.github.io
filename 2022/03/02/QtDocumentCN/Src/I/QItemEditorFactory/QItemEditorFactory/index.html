详细描述#

当在项目视图中编辑数据时，编辑器是由一个委托创建和显示的。QStyledItemDelegate是默认安装在Qt项目视图上的委托，它使用一个QItemEditorFactory来为它创建编辑器。一个由QItemEditorFactory提供的默认唯一实例被所有的项委托使用。如果你用setDefaultFactory()设置一个新的默认工厂，新工厂将被现有和新的委托使用。

一个工厂保存一个QItemEditorCreatorBase实例的集合，这些实例是专门的编辑器，为一个特定的QVariant数据类型产生编辑器（所有的Qt模型都以QVariant的形式存储它们的数据）。

标准编辑 Widget#

标准工厂实现为各种数据类型提供编辑器。每当一个委托人需要为模型提供的数据提供一个编辑器时，就会创建这些编辑器。下表显示了类型和提供的标准编辑器之间的关系。

类型 编辑器 Widget
bool QComboBox
double QDoubleSpinBox
int QSpinBox
无符号 int
QDate QDateEdit
QDateTime QDateTimeEdit
QPixmap QLabel
QString QLineEdit
QTime QTimeEdit

额外的编辑器可以用registerEditor()函数注册。

另请参阅 QStyledItemDelegate , 模型/视图编程 ，和 色彩编辑器工厂范例 .

void QItemEditorFactory:: registerEditor ( int userType , * creator )#

注册一个由creator指定的项目编辑器创建者，为给定userType的数据。

注意：工厂拥有项目编辑器创建者的所有权，如果以后有相同类型的新创建者被注册，它将被销毁。

另请参阅 createEditor ().

[static] void QItemEditorFactory:: setDefaultFactory ( * factory )#

将默认的项目编辑器工厂设置为给定的factory 。新的和现有的委托人都将使用新的工厂。

另请参阅 defaultFactory ().
