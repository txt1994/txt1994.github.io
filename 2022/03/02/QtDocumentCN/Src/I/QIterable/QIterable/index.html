成员函数文档编制#

bool QIterable:: canForwardIterate () const#

返回是否可以在容器上进行正向的迭代。这对应于容器的迭代器和const_iterator的std::forward_iterator_tag迭代器特质。

bool QIterable:: canInputIterate () const#

返回容器是否有一个输入迭代器。这与容器的迭代器和 const_iterator 的 std::input_iterator_tag 迭代器特性相对应。

bool QIterable:: canRandomAccessIterate () const#

返回是否有可能使用和迭代器有效地跳过多个值。这与容器的迭代器和const_iterator的std::random_access_iterator_tag迭代器特质相对应。

bool QIterable:: canReverseIterate () const#

返回是否可以对容器进行反向迭代。这与容器的 const_iterator 的 std::bidirectional_iterator_tag 迭代器特性相对应。

< Container > QIterable:: constBegin () const#

返回 QConstIterator为容器的开始。这可以用在stl风格的迭代中。

另请参阅 constEnd () 和 mutableBegin ().

< Container > QIterable:: constEnd () const#

返回一个Qterable::QConstIterator，用于容器的末端。这可以用在stl风格的迭代中。

另请参阅 constBegin () 和 mutableEnd ().

< Container > QIterable:: mutableBegin ()#

返回 QIterator 为容器的开始。这可以用在stl风格的迭代中。

另请参阅 mutableEnd () 和 constBegin ().

< Container > QIterable:: mutableEnd ()#

返回一个QSequentialIterable::iterator，用于容器的末端。这可以用在stl风格的迭代中。

另请参阅 mutableBegin () 和 constEnd ().

qsizetype QIterable:: size () const#

返回容器中的值的数量。
