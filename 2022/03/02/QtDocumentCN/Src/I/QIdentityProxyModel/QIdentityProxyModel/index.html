详细描述#

QIdentityProxyModel可以被用来准确地转发源模型的结构，不需要进行排序、过滤或其他转换。这在概念上类似于一个身份矩阵，其中A.I = A。

因为它不做任何排序或过滤，这个类最适合于转换源模型的数据()的代理模型。例如，可以创建一个代理模型来定义所使用的字体，或背景颜色，或工具提示等。这消除了在创建模型结构的同一个类中实现所有数据处理的需要，也可以用来创建可重用的组件。

这也提供了一种方法，在源模型由第三方提供而不能修改的情况下改变数据。

class DateFormatProxyModel : public QIdentityProxyModel
{
    // ...
    void setDateFormatString(const QString &formatString)
    {
        m_formatString = formatString;
    }
    QVariant data(const QModelIndex &index, int role) const override
    {
        if (role != Qt::DisplayRole)
            return QIdentityProxyModel::data(index, role);
        const QDateTime dateTime = sourceModel()->data(SourceClass::DateRole).toDateTime();
        return dateTime.toString(m_formatString);
    }
    private:
    QString m_formatString;
};

另请参阅 QAbstractProxyModel , 模型/视图编程 ，和 QAbstractItemModel .
