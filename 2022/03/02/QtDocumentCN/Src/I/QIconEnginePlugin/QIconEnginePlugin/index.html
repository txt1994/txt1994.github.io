详细描述#

图标引擎插件是一个简单的插件界面，它使创建自定义图标引擎变得容易，可以通过QIcon动态加载到应用程序中。QIcon使用文件或资源名称的后缀来决定使用何种图标引擎。

编写一个图标引擎插件是通过子类化这个基类，重新实现纯虚拟函数 create ()，和导出类采用 Q_PLUGIN_METADATA () 宏。

json元数据应该包含这个插件所支持的图标引擎键的列表。这些键对应于创建该插件时使用的文件或资源名称的后缀。键值不分大小写。

{ "Keys": [ "myiconengine" ] }

另请参阅 如何创建 Qt 插件 .
