# QInputSequence Class

class Qt3DInput ::QInputSequence

QInputSequence表示一组QAbstractActionInput，必须一个接一个地被触发。. 更多…

Header: #include
CMake: find_package(Qt6 COMPONENTS 3dinput REQUIRED) target_link_libraries(mytarget PRIVATE Qt6::3dinput)
qmake: QT += 3dinput
Since: Qt 5.7
实例化： InputSequence
继承： Qt3DInput::QAbstractActionInput

buttonInterval : int#

输入序列中连续的QAbstractActionInput之间的最大时间。时间的单位是毫秒。

访问函数：

timeout : int#

输入序列中所有QAbstractActionInput的时间必须被触发。时间的单位是毫秒。
