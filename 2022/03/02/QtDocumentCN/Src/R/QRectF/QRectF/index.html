详细描述#

矩形通常表示为一个左上角和一个尺寸。一个QRectF的尺寸（宽度和高度）总是等同于构成其渲染基础的数学矩形。

一个QRectF可以用一组左、上、宽和高的坐标来构造，或者从一个QPointF和一个QSizeF中构造。下面的代码创建了两个相同的矩形。

QRectF r1(100.0, 200.1, 11.2, 16.3)。
QRectF r2(QPointF(100.0, 200.1), QSizeF(11.2, 16.3))。

还有第三个构造函数从一个QRect创建一个QRectF，以及一个相应的toRect()函数，根据这个矩形的值返回一个QRect对象（注意，返回的矩形中的坐标被四舍五入为最近的整数）。

QRectF类提供了一个函数集合，用于返回各种矩形坐标，并能对这些坐标进行操作。QRectF还提供了相对于各种坐标来移动矩形的函数。此外，还有一个moveTo()函数可以移动矩形，将其左上角留在给定的坐标处。另外，translate()函数将矩形移动到相对于当前位置的给定偏移量，而translated()函数返回这个矩形的翻译副本。

size()函数以QSizeF的形式返回矩形的尺寸。也可以使用width()和height()函数单独检索尺寸。要操作尺寸，请使用setSize()、setWidth()或setHeight() 函数。另外，也可以通过应用设置矩形坐标的任一函数来改变尺寸，例如，setBottom()或setRight()。

contains()函数告诉人们一个给定的点是否在矩形内，而intersects()函数在这个矩形与一个给定的矩形相交时返回 “true”（否则为false）。QRectF类还提供了intersected()函数，用于返回相交的矩形，以及united()函数，用于返回包围给定矩形的矩形和这个。
相交() 相合()

如果矩形的宽度或高度小于或等于0，isEmpty()函数返回 “true”。 注意，空的矩形是无效的。如果宽度和高度都大于0，isValid()函数返回 “true”。另一方面，一个空的矩形（isNull() == true）的宽度和高度都设置为0。

注意，由于QRect和QRectF的定义方式，一个空的QRectF的定义方式与QRect基本相同。

最后，QRectF对象可以被流化，也可以被比较。

渲染#

当使用anti-aliased画图器时，QRectF的边界线将在数学矩形的边界线两边对称地呈现。但是当使用别名画师（默认）时，其他的规则就适用了。

那么，当用一个像素宽的笔渲染时，QRectF的边界线将被渲染到数学矩形边界线的右边和下面。

当用两像素宽的笔进行渲染时，边界线将被数学矩形从中间分开。只要笔的像素数设置为偶数，就会出现这种情况，而用奇数像素的笔渲染时，多余的像素将被渲染到数学矩形的右边和下面，就像一个像素的情况。

坐标#

QRectF类提供了一个函数集合，用于返回各种矩形坐标，并能对这些坐标进行操作。QRectF还提供了相对于各种坐标来移动矩形的函数。

例如：bottom()、setBottom()和moveBottom()函数。bottom()返回矩形底边的y坐标，setBottom()将矩形的底边设置为给定的y坐标（它可以改变高度，但绝不会改变矩形的顶边），moveBottom()垂直移动整个矩形，使矩形的底边位于给定的y坐标，其大小不变。

也可以使用adjust()函数给这个矩形的坐标添加偏移，以及使用adjusted()函数根据对原始矩形的调整来检索一个新的矩形。如果宽度和高度中的任何一个是负数，使用normalized()函数来检索一个角被调换的矩形。

此外，QRectF提供了getCoords()函数来提取矩形的左上角和右下角的位置，以及getRect()函数来提取矩形的左上角、宽和高。使用setCoords()和setRect()函数，可以一次性地操作矩形的坐标和尺寸。

See also QRect and QRegion.

bool QRectF::intersects(const &rectangle) const#

如果此矩形与给定的矩形相交（即它们之间有一个非空的重叠区域），则返回true'，否则返回false。

bool QRectF::isEmpty() const#

如果矩形是空的，返回 “true”，否则返回 “false”。

一个空的矩形有width() <= 0或height() <= 0。一个空的矩形是无效的（即isEmpty() == ! isValid()）。

使用normalized()函数来检索一个角被调换的矩形。

参见 isNull(), isValid(), 和 normalized()。

bool QRectF::isNull() const#

如果矩形是一个空的矩形，返回 “true”，否则返回 “false”。

一个空矩形的宽度和高度都被设置为0。一个空矩形也是空的，因此是无效的。

参见 isEmpty() 和 isValid()。

bool QRectF::isValid() const#

如果矩形是有效的，返回 “true”，否则返回 “false”。

一个有效的矩形具有width() > 0和height() > 0。注意，对于无效的矩形，没有定义像交集这样的非实质性操作。一个有效的矩形不是空的（即isValid() == ! isEmpty()）。

参见 isNull(), isEmpty(), 和 normalized()。

另见 top()和moveTop()。

void QRectF::setTopLeft(const &position)#

将矩形的左上角设置为给定的位置。可以改变大小，但绝不会改变矩形的右下角。

参见 topLeft() 和 moveTopLeft() 。

void QRectF::setTopRight(const &position)#

将矩形的右上角设置为给定的位置。可以改变大小，但绝不会改变矩形的左下角。

参见 topRight()和moveTopRight()。

void QRectF::setWidth( width)#

将矩形的宽度设置为给定的有限的width。右边的边缘会被改变，但左边的不会。

参见 width() 和 setSize()。

void QRectF::setX( x)#

将矩形的左边缘设置为给定的有限的x坐标。可以改变宽度，但绝不会改变矩形的右边缘。

相当于setLeft()。

参见 x(), setY(), 和 setTopLeft()。

void QRectF::setY( y)#

将矩形的上边缘设置为给定的有限的y坐标。可以改变高度，但绝不会改变矩形的底边。

相当于setTop()。

参见 y(), setX(), 和 setTopLeft()。

QRectF::size() const#

返回矩形的大小。

参见 setSize(), width(), and height()。

QRectF::toAlignedRect() const#

返回一个基于此矩形的值的QRect，该矩形是完全包含此矩形的最小的整数矩形。

也见 toRect()。

[自5.8]CGRect QRectF::toCGRect() const#

从一个QRectF创建一个CGRect。

这个函数是在Qt 5.8中引入的。

也见 fromCGRect()。

QRectF::toRect() const#

基于此矩形的值，返回一个QRect。注意，返回的矩形中的坐标被四舍五入到最近的整数。

也见 QRectF()和toAlignedRect()。

QRectF::top() const#

返回矩形顶部边缘的y坐标。相当于y()。

参见 setTop(), topLeft(), 和 topRight()。

QRectF::topLeft() const#

返回矩形的左上角的位置。

参见 setTopLeft(), top(), and left()。

QRectF::topRight() const#

返回矩形的右上角的位置。

参见 setTopRight(), top(), 和 right()。

void QRectF::translate( dx, dy)#

将矩形dx沿x轴移动，dy沿y轴移动，相对于当前位置。正值是将矩形向右和向下移动。两个参数都必须是有限的。

参见 moveTopLeft(), moveTo(), 和 translated()。

void QRectF::translate(const &offset)#

这是一个重载函数。

将矩形offset.x()沿x轴移动，offset.y()沿y轴移动，相对于当前位置。

QRectF::translate( dx, dy) const#

返回矩形的副本，相对于当前的位置，矩形被沿x轴平移dx，沿y轴平移dy。正值是将矩形向右和向下移动。两个参数都必须是有限的。

参见 translate()。

QRectF::translate(const &offset) const#

这是一个重载函数。

返回一个矩形的副本，该副本沿x轴被翻译offset.x()，沿y轴被翻译offset.y()，相对于当前位置。

[自5.7] QRectF::transposed() const#

返回一个交换了宽度和高度的矩形的副本。

QRectF r = {1.5, 5.1, 4.2, 2.4};
r = r.transposed(); // r == {1.5, 5.1, 2.4, 4.2}.

这个函数是在Qt 5.7中引入的。

也见 QSizeF::transposed()。

QRectF::united(const &rectangle) const#

返回这个矩形和给定的矩形的边界矩形。

又见 intersected()。

QRectF::width() const#

返回矩形的宽度。

又见 setWidth(), height(), and size()。

QRectF::x() const#

返回矩形的左边缘的x坐标。相当于left()。

参见 setX(), y(), 以及 topLeft()。

QRectF::y() const#

返回矩形顶部边缘的y坐标。相当于top()。

参见 setY(), x(), 和 topLeft()。

QRectF::operator&( const &rectangle) const#

返回这个矩形和给定的矩形的交点。如果没有交集，则返回一个空矩形。

另见 operator&=()和intersected()。

&QRectF::operator&=( const &rectangle)#

将此矩形与给定的矩形相交。

另见 intersected() 和 operator&()。

[自5.3] &QRectF::operator+=(const &margins)#

将margins添加到矩形中，使其增长。

这个函数是在Qt 5.3中引入的。

参见 marginsAdded(), marginsRemoved(), 和 operator-=().

[自5.3] &QRectF::operator-=(const &margins)#

返回一个由margins缩小的矩形。

这个函数是在Qt 5.3中引入的。

参见 marginsRemoved(), operator+=(), 和 marginsAdded()。

QRectF::operator|(const &rectangle) const#

返回这个矩形和给定的矩形的边界矩形。

也见 united()和operator|=()。

&QRectF::operator|=( const &rectangle)#

将此矩形与给定的矩形结合起来。

参见 united() 和 operator|()。

相关非成员#

bool operator! =(const &r1, const &r2)#

如果矩形r1和r2充分不同，则返回true，否则返回false。

警告：该函数不检查严格的不等式；相反，它使用模糊比较法来比较矩形的坐标。

[自5.3] operator+(const &lhs, const &rhs)#

返回由rhs边距增长的lhs矩形。

这个函数是在Qt 5.3中引入的。

[自5.3] operator+(const &lhs, const &rhs)#

这是一个重载函数。

返回由rhs边距增长的lhs矩形。

这个函数是在Qt 5.3中引入的。

[自5.3] operator-(const &lhs, const &rhs)#

返回由rhs边距缩小的lhs矩形。

这个函数是在Qt 5.3中引入的。

&operator<<( &stream, const &rectangle)#

将rectangle写入stream，并返回一个对该流的引用。

也见 序列化Qt数据类型。

bool operator==(const &r1, const &r2)#

如果矩形r1和r2是近似的，则返回 “true”，否则返回 “false”。

警告：该函数不检查严格的相等，而是使用模糊比较法来比较矩形的坐标。

参见 qFuzzyCompare。

&operator>>( &stream, &rectangle)#

从stream中读取一个矩形，并返回一个对该流的引用。

See also Serializing Qt Data Types.
