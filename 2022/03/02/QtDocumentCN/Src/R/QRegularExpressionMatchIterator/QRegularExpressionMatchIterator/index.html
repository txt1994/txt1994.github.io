# QRegularExpressionMatchIterator Class

QRegularExpressionMatchIterator类为QRegularExpression对象与字符串的全局匹配结果提供一个迭代器。 More…

详细描述#

QRegularExpressionMatchIterator对象是一个类似于Java的前向迭代器；它可以通过调用QRegularExpression::globalMatch()函数获得。一个新的QRegularExpressionMatchIterator将被定位在第一个结果之前。然后你可以调用hasNext()函数来检查是否有更多的结果可用；如果有，next()函数将返回下一个结果并推进迭代器。

每个结果都是一个QRegularExpressionMatch对象，持有该结果的所有信息（包括捕获的子字符串）。

比如说：

// extracts the words
QRegularExpression re("(\\w+)");
QString subject("the quick fox");
QRegularExpressionMatchIterator i = re.globalMatch(subject);
while (i.hasNext()) {
    QRegularExpressionMatch match = i.next();
    // ...
}

此外，QRegularExpressionMatchIterator提供了一个peekNext()函数来获取下一个结果，而不需要推进迭代器。

从Qt 6.0开始，也可以简单地在一个基于范围的for循环中使用QRegularExpression::globalMatch的结果，例如，像这样:

// using a raw string literal, R"(raw_characters)", to be able to use "\w"
// without having to escape the backslash as "\\w"
QRegularExpression re(R"(\w+)");
QString subject("the quick fox");
for (const QRegularExpressionMatch &match : re.globalMatch(subject)) {
    // ...
}

你可以通过调用 regularExpression() 函数来检索主题字符串被匹配的 QRegularExpression 对象；也可以通过调用 matchType() 和 matchOptions() 来获取匹配类型和匹配选项。

关于Qt正则表达式类的更多信息，请参考QRegularExpression文档。

See also QRegularExpression and QRegularExpressionMatch.

成员函数文档#

[since 6.1]QRegularExpressionMatchIterator::QRegularExpressionMatchIterator( &&iterator)#

通过从iterator移动，构造一个QRegularExpressionMatchIterator对象。

请注意，一个从QRegularExpressionMatchIterator移动过来的对象只能被销毁或被赋值。调用除销毁器或其中一个赋值运算符之外的其他函数的效果是未定义的。

这个函数是在Qt 6.1中引入的。

See also operator=().

QRegularExpressionMatchIterator::QRegularExpressionMatchIterator(const &iterator)#

构建一个QRegularExpressionMatchIterator对象，作为iterator的副本。

See also operator=().

[since 5.1]QRegularExpressionMatchIterator::QRegularExpressionMatchIterator()#

构建一个空的、有效的QRegularExpressionMatchIterator对象。正则表达式被设置为默认的构造；匹配类型为QRegularExpression::NoMatch，匹配选项为QRegularExpression::NoMatchOption。

在构建的对象上调用hasNext()成员函数将返回错误，因为迭代器没有在有效的匹配序列上进行迭代。

这个函数是在Qt 5.1中引入的。

&QRegularExpressionMatchIterator::operator=( &&iterator)#

Move-将iterator分配给这个对象，并返回一个对结果的引用。

请注意，一个从QRegularExpressionMatchIterator移动的对象只能被销毁或被分配。调用其他函数而不是销毁器或其中一个赋值运算符的效果是未定义的。

&QRegularExpressionMatchIterator::operator=(const &iterator)#

将迭代器iterator分配给这个对象，并返回一个对该副本的引用。

QRegularExpressionMatchIterator::~QRegularExpressionMatchIterator()#

销毁QRegularExpressionMatchIterator对象。

bool QRegularExpressionMatchIterator::hasNext() const#

如果在迭代器前面至少有一个匹配结果，则返回 “true”；否则返回 “false”。

See also next().

bool QRegularExpressionMatchIterator::isValid() const#

如果迭代器对象是由QRegularExpression::globalMatch()函数在一个有效的QRegularExpression对象上调用的结果，则返回true；如果QRegularExpression无效，则返回false。

See also QRegularExpression::globalMatch() and QRegularExpression::isValid().

QRegularExpressionMatchIterator::matchOptions() const#

返回用于获取此QRegularExpressionMatchIterator对象的匹配选项，也就是传递给QRegularExpression::globalMatch()的匹配选项。

See also QRegularExpression::globalMatch(), regularExpression(), and matchType().

QRegularExpressionMatchIterator::matchType() const#

返回用于获取此QRegularExpressionMatchIterator对象的匹配类型，也就是传递给QRegularExpression::globalMatch()的匹配类型。

See also QRegularExpression::globalMatch(), regularExpression(), and matchOptions().

QRegularExpressionMatchIterator::next()#

返回下一个匹配结果，并使迭代器前进一个位置。

注意：当迭代器处于结果集的末端时调用此函数会导致未定义的结果。

QRegularExpressionMatchIterator::peekNext() const#

在不移动迭代器的情况下返回下一个匹配结果。

注意：当迭代器处于结果集的末端时调用此函数会导致未定义的结果。

QRegularExpressionMatchIterator::regularExpression() const#

返回QRegularExpression对象，其globalMatch()函数返回此对象。

参见 QRegularExpression::globalMatch(), matchType(), 和 matchOptions()。

void QRegularExpressionMatchIterator::swap( &other)#

用这个迭代器对象交换迭代器other。这个操作非常快，而且从不失败。
