详细描述#

一个QRegularExpressionMatch对象可以通过调用QRegularExpression::match()函数获得，或者作为一个QRegularExpressionMatchIterator的全局匹配的单个结果。

匹配尝试的成功或失败可以通过调用hasMatch()函数进行检查。QRegularExpressionMatch还可以通过hasPartialMatch()函数报告部分匹配成功。

此外，QRegularExpressionMatch还返回模式字符串中由捕获组捕获的子字符串。索引为0的隐式捕获组捕获整个匹配的结果。captured()函数返回每个被捕获的子串，可以通过捕获组的索引或其名称。

QRegularExpression re("(\\d\\d) (?<name>\w+)")。
QRegularExpressionMatch match = re.match("23 Jordan")。
if (match.hasMatch()) {
    QString number = match.catch(1); // first == "23"
    QString name = match.capture("name"); // name == "Jordan"
}

对于每个被捕获的子串，可以通过调用captureStart()和captureEnd()函数，分别查询其在主题字符串中的起始和结束位置。每个被捕获的子串的长度可以通过captureLength()函数获得。

便捷函数capturedTexts()将按照捕获组捕获的顺序，一次性返回所有捕获的子串（包括整个模式所匹配的子串）；也就是说，captured(i) == capturedTexts().at(i)。

你可以通过调用regularExpression()函数来检索主题字符串被匹配的QRegularExpression对象；也可以通过调用matchType()和matchOptions()分别获取匹配类型和匹配选项。

关于Qt正则表达式类的更多信息，请参考QRegularExpression文档。

See also QRegularExpression.

[since 6.1]QRegularExpressionMatch::QRegularExpressionMatch( &&match)#

通过从给定的match移动结果来构造一个匹配结果。

请注意，一个从QRegularExpressionMatch移动过来的结果只能被销毁或被赋值。调用除析构器以外的其他函数或其中一个赋值运算符的效果是未定义的。

这个函数引入与Qt 6.1.

See also operator=().

QRegularExpressionMatch::QRegularExpressionMatch(const &match)#

通过复制给定的match的结果来构造一个匹配结果。

[since 5.1]QRegularExpressionMatch::QRegularExpressionMatch()#

构建一个有效的、空的QRegularExpressionMatch对象。正则表达式被设置为默认构建的表达式；匹配类型为QRegularExpression::NoMatch，匹配选项为QRegularExpression::NoMatchOption 。

该对象将通过hasMatch()和hasPartialMatch()成员函数报告没有匹配。

这个函数引入与Qt 5.1.

QRegularExpressionMatch::captured(int nth = 0) const#

返回第nth个捕获组所捕获的子串。

如果第nth个捕获组没有捕获一个字符串，或者没有这样的捕获组，则返回一个空的QString。

注意：隐含的捕获组编号为0，捕获整个模式所匹配的子串。

bool QRegularExpressionMatch::hasMatch() const#

如果正则表达式与主题字符串匹配，则返回 “true”，否则返回 “false”。

See also QRegularExpression::match() and hasPartialMatch().

bool QRegularExpressionMatch::hasPartialMatch() const#

如果正则表达式与主题字符串部分匹配，返回 “true”，否则返回 “false”。

注意：只有明确使用部分匹配类型之一的匹配才能产生部分匹配。尽管如此，如果这种匹配完全成功，这个函数将返回false，而hasMatch()将返回true。

See also QRegularExpression::match(), QRegularExpression::MatchType, and hasMatch().

bool QRegularExpressionMatch::isValid() const#

如果匹配对象是由QRegularExpression::match()函数在有效的QRegularExpression对象上调用的结果，则返回true；如果QRegularExpression无效，则返回false。

See also QRegularExpression::match() and QRegularExpression::isValid().

int QRegularExpressionMatch::lastCapturedIndex() const#

返回捕捉到的最后一个捕捉组的索引，包括隐含的捕捉组0。 这可以用来提取所有被捕捉的子字符串：

QRegularExpressionMatch match = re.match(string);
for (int i = 0; i <= match.lastCapturedIndex(); ++i) {
    QString captured = match.captured(i);
    // ...
}

请注意，一些索引小于lastCapturedIndex()的捕获组可能没有匹配，因此没有捕获任何东西。

如果正则表达式不匹配，该函数返回-1。

See also captured(), capturedStart(), capturedEnd(), and capturedLength().
