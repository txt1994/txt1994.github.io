详细描述#

矩形通常表示为一个左上角和一个尺寸。QRect的尺寸（宽度和高度）总是等同于构成其渲染基础的数学矩形。

一个QRect可以用一组左、上、宽和高的整数来构造，或者由一个QPoint和一个QSize来构造。下面的代码创建了两个相同的矩形。

QRect r1(100, 200, 11, 16);
QRect r2(QPoint(100, 200), QSize(11, 16));

有第三个构造函数可以使用左上角和右下角坐标创建QRect，但我们建议你避免使用它。理由是由于历史原因，bottom()和right()函数返回的值与矩形的真正右下角有偏差。

QRect类提供了一个函数集合，用于返回各种矩形坐标，并能对这些坐标进行操作。QRect还提供了相对于各种坐标来移动矩形的函数。此外，还有一个moveTo()函数可以移动矩形，将其左上角留在给定的坐标处。另外，translate()函数将矩形移动到给定的相对于当前位置的偏移量，而translated()函数返回这个矩形的一个翻译副本。

size()函数以QSize的形式返回矩形的尺寸。尺寸也可以用width()和height()函数单独检索。要操作尺寸，请使用setSize()、setWidth()或setHeight()函数。另外，也可以通过应用任何一个设置矩形坐标的函数来改变尺寸，例如，setBottom()或setRight()。

contains()函数告诉人们一个给定的点是否在矩形内，而intersects()函数如果这个矩形与一个给定的矩形相交，则返回true。QRect类还提供了intersected()函数，返回相交的矩形，以及united()函数，返回包围给定矩形和这个矩形的矩形:
intersected() united()

如果left()>right()或top()>bottom()，isEmpty()函数返回true'。注意，一个空的矩形是无效的。如果[left](#left)() <= [right](#right)() *和* [top](#top)() <= [bottom](#bottom)()，[isValid](#isValid)()函数返回true’。另一方面，一个空矩形（isNull() == true）的宽度和高度都被设置为0。

注意，由于QRect和QRectF的定义方式，一个空的QRect的定义方式与QRectF基本相同。

最后，QRect对象可以被流化，也可以被比较。

坐标#

QRect类提供了一个函数集合，用于返回各种矩形坐标，并能对这些坐标进行操作。QRect还提供了相对于各种坐标来移动矩形的函数。

例如left()、setLeft()和moveLeft()函数，作为一个例子。left()返回矩形左边缘的x坐标，setLeft()将矩形的左边缘设置为给定的x坐标（它可以改变宽度，但绝不会改变矩形的右边缘），moveLeft()将整个矩形水平移动，使矩形的左边缘位于给定的x坐标，其大小不变。

! img

请注意，由于历史原因，bottom()和right()函数返回的值与矩形的真正右下角有偏差。右](#right)()函数返回*左()+宽() - 1，底()函数返回顶()+高() - 1*。由bottomRight()便利函数返回的点的情况也是这样。此外，topRight()和bottomLeft()函数的x和y坐标，分别包含与真正的右边缘和底边缘相同的偏差。

我们建议你使用x()+width()和y()+height()来寻找真正的右下角，并避免使用right()和bottom()。另一个解决方案是使用QRectF。QRectF类使用浮点精度的坐标在平面内定义了一个矩形，而QRectF::right()和QRectF::bottom()函数*确实返回右和底的坐标。

也可以使用adjust()函数为这个矩形的坐标添加偏移量，以及使用adjusted()函数根据对原始矩形的调整检索一个新的矩形。如果宽度和高度中的任何一个是负数，使用normalized()函数来检索一个角被交换的矩形。

此外，QRect提供了getCoords()函数，可以提取矩形的左上角和右下角的位置，以及getRect()函数，提取矩形的左上角、宽度和高度。使用setCoords()和setRect()函数可以一次性地操作矩形的坐标和尺寸。

约束#

QRect被限制在 “int “类型的最小和最大值。对QRect的操作如果有可能导致超出这个范围的值，将导致未定义的行为。

See also QRectF and QRegion.

成员函数文档#

QRect::QRect(int x, int y, int width, int height)#

构建一个以（x, y）为左上角，给定宽和高的矩形。

See also setRect().

QRect::QRect(const &topLeft, const &size)#

构建一个具有给定topLeft角和给定size的矩形。

参见 setTopLeft()和setSize()。

QRect::QRect(const &topLeft, const &bottomRight)#

构建一个具有给定的topLeft和bottomRight角的矩形，两个角都包括在内。

如果bottomRight在topLeft的上方和左边，定义的矩形反而不包括角。

注意：要确保两个点都包括在内，无论相对顺序如何，请使用span()。

参见 setTopLeft(), setBottomRight(), 和 span()。

QRect::QRect()#

构建一个空矩形。

参见 isNull()。

void QRect::adjust(int dx1, int dy1, int dx2, int dy2)#

将dx1, dy1, dx2和dy2分别添加到矩形的现有坐标中。

参见 adjusted()和setRect()。

QRect::adjusted(int dx1, int dy1, int dx2, int dy2) const#

返回一个新的矩形，将dx1, dy1, dx2和dy2分别添加到这个矩形的现有坐标上。

参见 adjust()。

int QRect::bottom() const#

返回矩形的底边的Y坐标。

请注意，由于历史原因，这个函数返回top() + height() - 1；使用y() + height() 来检索真正的y坐标。

参见 setBottom(), bottomLeft(), 和bottomRight()。

QRect::bottomLeft() const#

返回矩形的左下角的位置。注意，由于历史原因，这个函数返回QPoint(left(), top() + height() - 1)。

另见 setBottomLeft(), bottom(), 和 left()。

QRect::bottomRight() const#

返回矩形的右下角的位置。

请注意，由于历史原因，这个函数返回QPoint(left() + width() -1, top() + height() -1) 。

另见 setBottomRight(), bottom(), 和 right()。

QRect::center() const#

返回矩形的中心点。

See also moveCenter().

bool QRect::contains(const &point, bool proper = false) const#

如果给定的点在矩形的内部或边缘上，则返回 “true”，否则返回 “false”。如果proper为真，这个函数只在给定的点在矩形的*内（即不在边缘）时返回`真’。

参见 intersects()。

bool QRect::contains(const &rectangle, bool proper = false) const#

这是一个重载函数。

如果给定的rectangle在这个矩形内，则返回 “true”，否则返回 “false”。如果proper为真，这个函数只在矩形完全在这个矩形内（不在边缘上）时返回`真’。

bool QRect::contains(int x, int y) const#

这是一个重载函数。

如果点(x, y)在这个矩形内，返回 “true”，否则返回 “false”。

bool QRect::contains(int x, int y, bool proper) const#

这是一个重载函数。

如果点(x, y)在矩形的内部或边缘上，则返回 “true”，否则返回 “false”。如果proper为真，该函数只在点完全在矩形内（不在边缘）时返回true。

提取矩形的左上角的位置为x1*和y1，右下角的位置为**x2和*y2。

参见 setCoords()和getRect()。

void QRect::getRect(int *x, int *y, int *width, int *height) const#

提取矩形左上角的位置为x*和y，其尺寸为**width和*height。

int QRect::height() const#

返回矩形的高度。

参见 setHeight(), width, and size()。

QRect::intersected(const &rectangle) const#

返回这个矩形和给定的矩形的交集。注意，r.intersected(s)等同于r & s。

参见 intersects(), united(), 和 operator&=()。

bool QRect::intersects(const &rectangle) const#

如果此矩形与给定的矩形相交，则返回 “true”（即，至少有一个像素在两个矩形内），否则返回 “false”。

相交的矩形可以用intersected()函数来检索。

也见 contains()。

bool QRect::isEmpty() const#

如果矩形是空的，返回 “true”，否则返回 “false”。

一个空的矩形有一个left() > right() 或 top() > bottom()。一个空的矩形是无效的（即，isEmpty() == isValid()）。

使用normalized()函数来检索一个边角互换的矩形。

参见 isNull(), isValid(), 和 normalized()。

bool QRect::isNull() const#

如果矩形是一个空的矩形，返回 “true”，否则返回 “false”。

一个空矩形的宽度和高度都被设置为0（即，right() == left() - 1 和 bottom() == top() - 1）。一个空的矩形也是空的，因此是无效的。

参见 isEmpty()和isValid()。

bool QRect::isValid() const#

如果矩形是有效的，返回 “true”，否则返回 “false”。

一个有效的矩形具有left() <= right() 和 top() <= bottom() 。请注意，像交集这样的非琐碎操作并没有为无效矩形定义。一个有效的矩形不是空的（也就是说，isValid() == ()）。

参见 isNull(), isEmpty(), 和 normalized()。

返回矩形的左边缘的X坐标。相当于x()。

参见 setLeft(), topLeft(), and bottomLeft()。

QRect::normalized() const#

返回一个规范化的矩形；也就是说，一个具有非负的宽度和高度的矩形。

如果width() < 0，该函数将交换左角和右角，如果height() < 0，它将交换顶角和底角，同时将角从非包容性改为包容性。

参见 isValid()和isEmpty()。

参见 setRight(), topRight(), 和 bottomRight()。

void QRect::setLeft(int x)#

将矩形的左边缘设置为给定的x坐标。可以改变宽度，但绝不会改变矩形的右边缘。

相当于setX()。

另见 left()和moveLeft()。

void QRect::setRect(int x, int y, int width, int height)#

设置矩形的左上角坐标为(x, y)，其大小为给定的width和height。

参见 getRect()和setCoords()。

void QRect::setRight(int x)#

将矩形的右边缘设置为给定的x坐标。可以改变宽度，但绝不会改变矩形的左边缘。

另见 right()和moveRight()。

void QRect::setSize(const &size)#

将矩形的大小设置为给定的size。左上角不会被移动。

参见 size(), setWidth(), and setHeight()。

void QRect::setTop(int y)#

将矩形的顶部边缘设置为给定的y坐标。可以改变高度，但绝不会改变矩形的底边。

相当于setY()。

另见 top()和moveTop()。

void QRect::setTopLeft(const &position)#

将矩形的左上角设置为给定的位置。可以改变大小，但绝不会改变矩形的右下角。

参见 topLeft()和moveTopLeft()。

void QRect::setTopRight(const &position)#

将矩形的右上角设置为给定的位置。可以改变大小，但绝不会改变矩形的左下角。

参见 topRight()和moveTopRight()。

void QRect::setWidth(int width)#

将矩形的宽度设置为给定的宽度。右边的边缘会被改变，但左边的不会。

参见 width() 和 setSize()。

void QRect::setX(int x)#

将矩形的左边缘设置为给定的x坐标。可以改变宽度，但绝不会改变矩形的右边缘。

相当于setLeft()。

参见 x(), setY(), 以及 setTopLeft()。

void QRect::setY(int y)#

将矩形的上边缘设置为给定的y坐标。可以改变高度，但绝不会改变矩形的底边。

相当于setTop()。

参见 y(), setX(), and setTopLeft()。

QRect::size() const#

返回矩形的大小。

参见 setSize(), width(), and height()。

[静态，自6.0起] QRect::span(const &p1, const &p2)#

返回一个横跨两个点p1和p2的矩形，包括这两个点和中间的所有东西。

这个函数是在Qt 6.0中引入的。

[自5.8]CGRect QRect::toCGRect() const#

从一个QRect创建一个CGRect。

这个函数是在Qt 5.8中引入的。

参见 QRectF::fromCGRect()。

int QRect::top() const#

返回矩形顶部边缘的Y坐标。相当于y()。

参见 setTop(), topLeft(), and topRight()。

QRect::topLeft() const#

返回矩形的左上角的位置。

参见 setTopLeft(), top(), 和 left。

QRect::topRight() const#

返回矩形的右上角的位置。

注意，由于历史原因，这个函数返回QPoint(left() + width() -1, top()。

另见 setTopRight(), top(), 和 right()。

void QRect::translate(int dx, int dy)#

将矩形dx沿x轴移动，dy沿y轴移动，相对于当前位置。正值是将矩形向右和向下移动。

参见 moveTopLeft(), moveTo(), 和 translated()。

void QRect::translate(const &offset)#

这是一个重载函数。

将矩形offset.x()沿x轴移动，offset.y()沿y轴移动，相对于当前位置。

QRect::translate(int dx, int dy) const#

返回一个矩形的副本，该副本沿x轴平移dx，沿y轴平移dy，相对于当前位置。正值将矩形向右和向下移动。

另见 translate()。

QRect::translate(const &offset) const#

这是一个重载函数。

返回一个矩形的副本，该副本沿x轴平移了offset.x()，沿y轴平移了offset.y()，相对于当前位置。

[自5.7] QRect::transposed() const#

返回一个交换了宽度和高度的矩形的副本。

QRect r = {15, 51, 42, 24};
r = r.transposed(); // r == {15, 51, 24, 42}。

这个函数是在Qt 5.7中引入的。

参见 QSize::transposed()。
