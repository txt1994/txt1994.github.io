详细描述#

QByteArrayList 实际上只是一个 QList<QByteArray >。它被记录为一个完整的类，只是为了简单地记录仅存在于 QList<[QByteArray](https:// doc.qt.io/qt-6/qbytearray.html)>。

QList 的所有功能也适用于 QByteArrayList。例如可以使用isEmpty()来测试列表是否为空，可以调用append)(), 前置(), 插入()、替换(), removeAll(), [removeAt](https://doc.qt.io/qt-6/qlist .html#removeAt)(), removeFirst(), [removeLast](https://doc.qt.io/qt -6/qlist.html#removeLast)() 和 removeOne() 来修改 QByteArrayList。此外，QByteArrayList 提供了几个 join() 方法用于将列表连接成单个 [QByteArray](https://doc. qt.io/qt-6/qbytearray.html）。

QByteArrayList 的用途与 QStringList 的用途有很大不同。尽管 QStringList 有许多用于操作列表中元素的方法，但 QByteArrayList 没有。通常，只要使用可打印字符串列表，就应该使用 QStringList。 QByteArrayList 应该用于处理和有效地连接大量二进制数据，例如通过 QIODevice 顺序接收序列化数据时。

See also QByteArray and QStringList.

QByteArrayList 提供了 [Java 风格的迭代器](https://doc.qt.io/qt-6/java-style-iterators .html#java-style-iterators）和[STL-style iterators]（https://doc.qt.io/qt-6/containers.html#stl-style-iterators）。 Java 风格的 const 迭代器只是 QListIterator<QByteArray 的类型定义 -6/qbytearray.html)>.
