详细描述#

QBitmap 类是一个单色离屏绘制设备，主要用于创建自定义 QCursor 和 [QBrush](https://doc.qt .io/qt-6/qbrush.html) 对象，构造 QRegion 对象，以及为像素图和小部件设置掩码。

QBitmap 是一个 QPixmap 子类，确保深度为 1，但深度为 0 的空对象除外。如果像素图的深度更大比 1 分配给位图，位图将自动抖动。

使用 QColor 对象 [Qt::color0](https://doc.qt.io/qt-6/qt.html# GlobalColor-enum) 和 Qt::color1 在 QBitmap 对象（或 [QPixmap](https:// /doc.qt.io/qt-6/qpixmap.html) 深度为 1) 的对象。

使用 Qt::color0 绘画将位图位设置为 0，并使用 [Qt::color1](https: //doc.qt.io/qt-6/qt.html#GlobalColor-enum) 将位设置为 1。对于位图，0 位表示背景（或透明像素），1 位表示前景（或不透明像素） ）。使用 clear() 函数将所有位设置为 [Qt::color0](https://doc.qt. io/qt-6/qt.html#GlobalColor-enum）。请注意，使用 Qt::black 和 [Qt::white](https://doc.qt.io /qt-6/qt.html#GlobalColor-enum) 颜色没有意义，因为 QColor::pixel() 值不一定是黑色的 0 和白色的 1。

QBitmap 类提供了 transformed() 函数，返回位图的转换副本；使用 QTransform 参数来平移、缩放、剪切和旋转位图。此外，QBitmap 提供了静态 fromData() 函数，该函数返回从给定的 uchar 数据构造的位图，静态fromImage() 函数返回 QImage 的转换副本/qimage.html) 对象。

就像 QPixmap 类一样，QBitmap 通过使用隐式数据共享进行了优化。有关详细信息，请参阅 隐式数据共享 文档。

也可以看看 QPixmap, QImage, QImageReader, and QImageWriter.

成员函数文档#

QBitmap::QBitmap(const &fileName, const char *format = nullptr)#

从给定的 fileName 指定的文件构造位图。 如果文件不存在或格式未知，则位图将变为空位图。

fileName 和 format 参数被传递给 QPixmap::load() 函数。 如果文件格式每像素使用超过 1 位，则生成的位图将自动抖动。

也可以看看 QPixmap::isNull() and QImageReader::imageFormat().

QBitmap::QBitmap(int width, int height)#

用给定的 width 和 height 构造一个位图。 里面的像素是未初始化的。

也可以看看 clear().

QBitmap::QBitmap()#

构造一个空位图。

也可以看看 QPixmap::isNull().

void QBitmap::clear()#

清除位图，将其所有位设置为 Qt::color0。

[static] QBitmap::fromData(const &size, const *bits, monoFormat = QImage::Format_MonoLSB)#

用给定的 size 构造一个位图，并将内容设置为提供的 bits。

位图数据必须是字节对齐的，并以 monoFormat 指定的位顺序提供。 单声道格式必须是 QImage::Format_Mono 或 [QImage::Format_MonoLSB](https://doc.qt .io/qt-6/qimage.html#Format-enum）。 使用 QImage::Format_Mono 指定 XBM 格式的数据。

也可以看看 fromImage().

[static] QBitmap::fromImage(const &image, flags = Qt::AutoColor)#

返回使用指定图像转换 flags 转换为位图的给定 image 的副本。

也可以看看 fromData().

[static, since 5.12] QBitmap::fromImage( &&image, flags = Qt::AutoColor)#

这是一个过载功能。

返回使用指定图像转换 flags 转换为位图的给定 image 的副本。

这个函数是在 Qt 5.12 中引入的。

也可以看看 fromData().

[static] QBitmap::fromPixmap(const &pixmap)#

返回转换为位图的给定 pixmap 的副本。

如果像素图的深度大于 1，则生成的位图将自动抖动。

也可以看看 QPixmap::depth().

void QBitmap::swap( &other)#

用此位图交换位图 other。 此操作非常快且永不失败。

QBitmap::transformed(const &matrix) const#

返回此位图的副本，根据给定的 matrix 进行转换。

也可以看看 QPixmap::transformed().

QBitmap::operator QVariant() const#

以 QVariant 形式返回位图。
