详细描述#

QBitArray 是一个数组，可以访问各个位并提供运算符AND、[OR](https:// doc.qt.io/qt-6/qbitarray.html#operator-7c)、XOR和NOT)适用于整个位数组。 它使用 隐式共享 (copy-on-write) 来减少内存使用并避免不必要的数据复制。

以下代码构造了一个包含 200 位初始化为 false (0) 的 QBitArray：

QBitArray ba(200);

要将这些位初始化为 true，请将 true 作为第二个参数传递给构造函数，或者稍后调用 fill()。

QBitArray 使用从 0 开始的索引，就像 C++ 数组一样。 要访问特定索引位置的位，可以使用 operator。 在非常量位数组上，operator 返回对可用于赋值左侧的位的引用。 例如：

QBitArray ba;
ba.resize(3);
ba[0] = true;
ba[1] = false;
ba[2] = true;

由于技术原因，使用 testBit() 和 setBit() 效率更高访问数组中的位而不是 operator。 例如：

QBitArray ba(3);
ba.setBit(0, true);
ba.setBit(1, false);
ba.setBit(2, true);

QBitArray 支持 & (AND)、| ([OR](https://doc.qt. io/qt-6/qbitarray.html#operator-7c)), ^ (XOR), ~ (NOT)，以及 &=、|= 和 ^=。 这些运算符的工作方式与内置的同名 C++ 位运算符相同。 例如：

QBitArray x(5);
x.setBit(3, true);
// x: [ 0, 0, 0, 1, 0 ]

QBitArray y(5);
y.setBit(4, true);
// y: [ 0, 0, 0, 0, 1 ]

x |= y;
// x: [ 0, 0, 0, 1, 1 ]

由于历史原因，QBitArray 区分了空位数组和空位数组。 null 位数组是使用 QBitArray 的默认构造函数初始化的位数组。 empty 位数组是大小为 0 的任何位数组。空位数组始终为空，但空位数组不一定为空：

QBitArray().isNull();           // returns true
QBitArray().isEmpty();          // returns true

QBitArray(0).isNull();          // returns false
QBitArray(0).isEmpty();         // returns true

QBitArray(3).isNull();          // returns false
QBitArray(3).isEmpty();         // returns false

除 isNull() 之外的所有函数都将空位数组视为空位数组； 例如，QBitArray() 比较等于 QBitArray(0)。 我们建议您始终使用 isEmpty() 并避免使用 [isNull](https://doc.qt.io/qt- 6/qbitarray.html#isNull)()。

See also QByteArray and QList.

成员函数文档#

[since 5.2]QBitArray::QBitArray( &&other)#

移动构造一个 QBitArray 实例，使其指向 other 指向的同一个对象。

这个函数是在 Qt 5.2 中引入的。

QBitArray::QBitArray(const &other)#

构造 other 的副本。

这个操作需要恒定时间，因为QBitArray是[隐式共享](https://doc.qt.io/qt -6/隐式共享.html）。 这使得从函数返回 QBitArray 非常快。 如果共享实例被修改，它将被复制（写时复制），这需要[线性时间]（https://doc.qt.io/qt-6/containers.html#linear-time）。

See also operator=().

QBitArray::QBitArray(qsizetype size, bool value = false)#

构造一个包含 size 位的位数组。 这些位使用 value 进行初始化，默认为 false (0)。

QBitArray::QBitArray()#

构造一个空位数组。

See also isEmpty().

[since 5.2] &QBitArray::operator=( &&other)#

将 other 移动到该位数组并返回对该位数组的引用。

这个函数是在 Qt 5.2 中引入的。

&QBitArray::operator=(const &other)#

将 other 分配给该位数组并返回对该位数组的引用。

bool QBitArray::at(qsizetype i) const#

返回索引位置 i 处的位值。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

See also [operator]().

[since 5.11]const char *QBitArray::bits() const#

返回指向此 QBitArray 的密集位数组的指针。 位从每个字节中的最低有效位向上计数。 最后一个字节中相关的位数由“size() % 8”给出。

这个函数是在 Qt 5.11 中引入的。

See also fromBits() and size().

void QBitArray::clear()#

清除位数组的内容并使其为空。

See also resize() and isEmpty().

void QBitArray::clearBit(qsizetype i)#

将索引位置 i 处的位设置为 0。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

See also setBit() and toggleBit().

qsizetype QBitArray::count() const#

与 size() 相同。

qsizetype QBitArray::count(bool on) const#

如果 on 为真，则此函数返回位数组中存储的 1 位数； 否则返回 0 位数。

bool QBitArray::fill(bool value, qsizetype size = -1)#

将位数组中的每一位设置为value，如果成功则返回true； 否则返回“假”。 如果 size 不同于 -1（默认值），则位数组的大小会预先调整为 size。

例子：

QBitArray ba(8);
ba.fill(true);
// ba: [ 1, 1, 1, 1, 1, 1, 1, 1 ]

ba.fill(false, 2);
// ba: [ 0, 0 ]

See also resize().

void QBitArray::fill(bool value, qsizetype begin, qsizetype end)#

这是一个过载功能。

将索引位置 begin 直到（但不包括）end 处的位设置为 value。

begin 必须是位数组中的有效索引位置 (0 <= begin < size())。

end 必须是有效的索引位置或等于 size()，在这种情况下，填充操作将一直运行到结束 数组（0 <= end <= size()）。

例子：

QBitArray ba(4);
ba.fill(true, 1, 2);            // ba: [ 0, 1, 0, 0 ]
ba.fill(true, 1, 3);            // ba: [ 0, 1, 1, 0 ]
ba.fill(true, 1, 4);            // ba: [ 0, 1, 1, 1 ]

[static, since 5.11] QBitArray::fromBits(const char *data, qsizetype size)#

创建一个 QBitArray，其密集位数组位于 data，具有 size 位。 data 处的字节数组必须至少为 size / 8（向上取整）字节长。

如果 size 不是 8 的倍数，则此函数将包含 data 中最后一个字节的最低 size % 8 位。

这个函数是在 Qt 5.11 中引入的。

See also bits().

bool QBitArray::isEmpty() const#

如果此位数组的大小为 0，则返回 true； 否则返回假。

See also size().

bool QBitArray::isNull() const#

如果此位数组为空，则返回 true； 否则返回“假”。

例子：

QBitArray().isNull();           // returns true
QBitArray(0).isNull();          // returns false
QBitArray(3).isNull();          // returns false

由于历史原因，Qt 对空位数组和空位数组进行了区分。 对于大多数应用程序来说，重要的是位数组是否包含任何数据，这可以使用 isEmpty() 来确定。

See also isEmpty().

void QBitArray::resize(qsizetype size)#

将位数组的大小调整为 size 位。

如果 size 大于当前大小，则扩展位数组以使其成为 size 位，并在末尾添加额外的位。 新位被初始化为假 (0)。

如果 size 小于当前大小，则从末尾删除位。

See also size().

void QBitArray::setBit(qsizetype i)#

将索引位置 i 处的位设置为 1。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

See also clearBit() and toggleBit().

void QBitArray::setBit(qsizetype i, bool value)#

这是一个过载功能。

将索引位置 i 处的位设置为 value。

qsizetype QBitArray::size() const#

返回存储在位数组中的位数。

See also resize().

void QBitArray::swap( &other)#

用此位数组交换位数组 other。 此操作非常快且永不失败。

bool QBitArray::testBit(qsizetype i) const#

如果索引位置 i 处的位为 1，则返回 true； 否则返回“假”。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

See also setBit() and clearBit().

[since 6.0] QBitArray::toUInt32( endianness, bool *ok = nullptr) const#

返回转换为 int 的位数组。 转换基于 endianness。 将数组的前 32 位转换为 quint32 并返回它，遵循 endianness。 如果 ok 不是空指针，并且数组超过 32 位，则 ok 设置为 false，此函数返回零； 否则，它设置为真。

此功能是在 Qt 6.0 中引入的。

bool QBitArray::toggleBit(qsizetype i)#

反转索引位置 i 处的位的值，将该位的先前值返回为真（如果已设置）或假（如果未设置）。

如果之前的值为 0，则新的值为 1。如果之前的值为 1，则新的值为 0。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

See also setBit() and clearBit().

void QBitArray::truncate(qsizetype pos)#

截断索引位置 pos 处的位数组。

如果 pos 超出数组的末尾，则不会发生任何事情。

See also resize().

bool QBitArray::operator!=(const &other) const#

如果 other 不等于该位数组，则返回 true； 否则返回“假”。

See also operator==().

&QBitArray::operator&=(const &other)#

在此位数组中的所有位与 other 之间执行 AND 运算。 将结果分配给此位数组，并返回对它的引用。

结果具有两个位数组中最长的长度，任何丢失的位（如果一个数组比另一个数组短）被视为 0。

例子：

QBitArray a(3);
QBitArray b(2);
a[0] = 1; a[1] = 0; a[2] = 1;   // a: [ 1, 0, 1 ]
b[0] = 1; b[1] = 1;             // b: [ 1, 1 ]
a &= b;                         // a: [ 1, 0, 0 ]

See also operator&(), operator|=(), operator^=(), and operator~().

bool QBitArray::operator==(const &other) const#

如果 other 等于该位数组，则返回 true； 否则返回“假”。

See also operator!=().

QBitRef QBitArray::operator[](qsizetype *i*)#

返回索引位置 i 处的位作为可修改的引用。

i 必须是位数组中的有效索引位置（即 0 <= i < size()） .

例子：

QBitArray a(3);
a[0] = false;
a[1] = true;
a[2] = a[0] ^ a[1];

返回值是 QBitRef 类型，是 QBitArray 的辅助类。 当你得到一个 QBitRef 类型的对象时，你可以给它赋值，赋值将应用于你从中得到的 QBitArray 中的位 得到了参考。

函数 testBit()、[setBit](https://doc.qt.io/qt-6/qbitarray.html #setBit)() 和 clearBit() 稍快一些。

See also at(), testBit(), setBit(), and clearBit().

bool QBitArray::operator[](qsizetype *i*) const#

这是一个过载功能。

&QBitArray::operator^=(const &other)#

在此位数组中的所有位与 other 之间执行 XOR 操作。 将结果分配给此位数组，并返回对它的引用。

结果具有两个位数组中最长的长度，任何丢失的位（如果一个数组比另一个数组短）被视为 0。

例子：

QBitArray a(3);
QBitArray b(2);
a[0] = 1; a[1] = 0; a[2] = 1;   // a: [ 1, 0, 1 ]
b[0] = 1; b[1] = 1;             // b: [ 1, 1 ]
a ^= b;                         // a: [ 0, 1, 1 ]

See also operator^(), operator&=(), operator|=(), and operator~().

&QBitArray::operator|=(const &other)#

在此位数组中的所有位与 other 之间执行 OR 运算。 将结果分配给此位数组，并返回对它的引用。

结果具有两个位数组中最长的长度，任何丢失的位（如果一个数组比另一个数组短）被视为 0。

例子：

QBitArray a(3);
QBitArray b(2);
a[0] = 1; a[1] = 0; a[2] = 1;   // a: [ 1, 0, 1 ]
b[0] = 1; b[1] = 1;             // b: [ 1, 1 ]
a |= b;                         // a: [ 1, 1, 1 ]

See also operator|(), operator&=(), operator^=(), and operator~().

QBitArray::operator~() const#

返回包含此位数组的反转位的位数组。

例子：

QBitArray a(3);
QBitArray b;
a[0] = 1; a[1] = 0; a[2] = 1;   // a: [ 1, 0, 1 ]
b = ~a;                         // b: [ 0, 1, 0 ]

See also operator&(), operator|(), and operator^().

operator&(const &a1, const &a2)#

返回一个位数组，它是位数组 a1 和 a2 的与。

结果具有两个位数组中最长的长度，任何丢失的位（如果一个数组比另一个数组短）被视为 0。

QBitArray a(3);

c = a & b;                      // c: [ 1, 0, 0 ]

See also operator&=(), operator|(), and operator^().

See also QBitArray::operator|=(), operator&(), and operator^().
